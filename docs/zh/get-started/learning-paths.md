通过 [n8n Academy](https://learn.n8n.io/) 的交互式课程学习 n8n。你可以构建真实工作流、获得徽章，并取得结课证书。

所有课程都包括：

- 动手构建工作流的练习
- 章节测验和期末考试
- 通过后获得徽章和证书（及格分数为 70%）
- 练习用 API（大多数课程不需要外部账号）

完整课程列表请参阅 [n8n Academy 课程](https://learn.n8n.io/courses)。
