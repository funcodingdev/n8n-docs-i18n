# 使用社区 node <a href="#using-community-nodes" id="using-community-nodes"></a>

要使用社区 node，你首先需要[安装](installation-and-management/README.md)它们。

## 将社区 node 添加到 workflow <a href="#adding-community-nodes-to-your-workflow" id="adding-community-nodes-to-your-workflow"></a>

安装社区 node 后，你可以像使用其他 node 一样使用它。n8n 会在 **Nodes** 面板的搜索结果中显示该 node。n8n 会在 nodes panel 中用 **Package** <img src="../.gitbook/assets/package.png" alt="Package icon" data-size="line"> 图标标记社区 node。

## 名称重复的社区 node <a href="#community-nodes-with-duplicate-names" id="community-nodes-with-duplicate-names"></a>

多个社区 node 可能具有相同名称。如果在 workflow 中使用两个同名 node，除非它们有不同图标，否则它们看起来会一样。
