# n8n 社区 node 封禁列表 <a href="#n8n-community-node-blocklist" id="n8n-community-node-blocklist"></a>

n8n 维护了一份社区 node 封禁列表。你无法安装此列表中的任何 node。

n8n 可能会出于多种原因将社区 node 加入封禁列表，包括：

* 该 node 存在有意恶意行为
* 质量较低（低到可能造成危害）

如果你是社区 node 创建者，你的 node 位于封禁列表中，并且你认为这是误判，请联系 [hello@n8n.io](mailto: hello@n8n.io)。
