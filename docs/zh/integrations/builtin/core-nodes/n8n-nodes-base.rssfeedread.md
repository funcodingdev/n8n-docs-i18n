# RSS Read <a href="#rss-read" id="rss-read"></a>

使用 RSS Read node 读取互联网上发布的 RSS feed 数据。

## Node 参数 <a href="#node-parameters" id="node-parameters"></a>

- **URL**：输入要读取的 RSS 发布源 URL。

## Node 选项 <a href="#node-options" id="node-options"></a>

* **Ignore SSL Issues**：选择 n8n 是否应忽略 SSL/TLS 验证（开启），或不忽略（关闭）。
