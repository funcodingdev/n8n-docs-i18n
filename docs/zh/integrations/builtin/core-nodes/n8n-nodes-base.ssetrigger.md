# SSE Trigger node <a href="#sse-trigger-node" id="sse-trigger-node"></a>

Server-Sent Events（SSE）是一种服务器推送技术，让客户端可以通过 HTTP 连接接收来自服务器的自动更新。SSE Trigger node 用于接收服务器发送事件。

## Node parameters <a href="#node-parameters" id="node-parameters"></a>

SSE Trigger node 有一个参数：**URL**。输入要从中接收服务器发送事件（SSE）的 URL。
