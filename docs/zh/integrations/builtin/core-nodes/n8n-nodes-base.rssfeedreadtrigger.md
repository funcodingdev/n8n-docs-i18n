# RSS Feed Trigger node <a href="#rss-feed-trigger-node" id="rss-feed-trigger-node"></a>

当新的 RSS feed item 发布时，RSS Feed Trigger node 可用于启动 n8n workflow。

在本页中，你可以找到 RSS Feed Trigger node 支持的操作列表，以及更多资源链接。

## Node 参数 <a href="#node-parameters" id="node-parameters"></a>

* **Poll Times**：选择轮询 **Mode**，以设置触发轮询的频率。你选择的 **Mode** 会添加或移除相关字段。请参阅下方各节，配置每种 mode 类型的参数。
* **Feed URL**：输入要轮询的 RSS feed URL。
