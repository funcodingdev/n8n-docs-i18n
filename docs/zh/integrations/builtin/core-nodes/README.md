# Core nodes 库 <a href="#core-nodes-library" id="core-nodes-library"></a>

本节介绍 n8n 的 core nodes[^1]。

[^1]: 在 n8n 中，node 是用于组合 workflow 的独立组件。Node 定义 workflow 何时运行，允许你获取、发送和处理数据，可以定义流程控制逻辑，并连接外部服务。
