# No Operation, do nothing <a href="#no-operation-do-nothing" id="no-operation-do-nothing"></a>

当你不想执行任何操作时，可以使用 No Operation, do nothing node。此 node 的目的是让 workflow 更易阅读，并清楚标示数据流在哪里停止。这可以帮助其他人从视觉上更好地理解 workflow。
