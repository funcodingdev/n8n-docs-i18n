# PhantomBuster credentials <a href="#phantombuster-credentials" id="phantombuster-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [PhantomBuster](../app-nodes/n8n-nodes-base.phantombuster.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [PhantomBuster](https://www.phantombuster.com/) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [PhantomBuster's API documentation](https://hub.phantombuster.com/reference)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- 一个 **API Key**：要获取 API key，请前往 [**Workspace settings**](https://phantombuster.com/workspace-settings) **> Third party API keys**，并选择 **+ Add API Key**。更多信息请参考 [How to find my API key](https://hub.phantombuster.com/docs/api#how-to-find-my-api-key)。
