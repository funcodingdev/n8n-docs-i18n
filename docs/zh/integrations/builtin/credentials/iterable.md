# Iterable credentials <a href="#iterable-credentials" id="iterable-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [Iterable](../app-nodes/n8n-nodes-base.iterable.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Iterable](https://iterable.com) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 Iterable 的 API documentation：

- [US-based Iterable projects](https://api.iterable.com/api/docs)
- [Europe-based Iterable projects](https://api.eu.iterable.com/api/docs)

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- **API Key**：创建 API keys 的说明请参考 [Iterable's Creating API keys documentation](https://support.iterable.com/hc/en-us/articles/360043464871-API-Keys#creating-api-keys)。
