# Mindee credentials <a href="#mindee-credentials" id="mindee-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [Mindee](../app-nodes/n8n-nodes-base.mindee.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Mindee](https://mindee.com) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- Invoice API key：用于 [Invoice OCR API](https://www.mindee.com/product/invoice-ocr-api)
- Receipt API key：用于 [Receipt OCR API](https://www.mindee.com/product/receipt-ocr-api)

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关各项服务的更多信息，请参考 [Mindee's Invoice OCR API documentation](https://developers.mindee.com/docs/invoice-ocr) 和 [Mindee's Receipt OCR API documentation](https://developers.mindee.com/docs/receipt-ocr)。

## 使用 invoice API key <a href="#using-invoice-api-key" id="using-invoice-api-key"></a>

要配置此 credential，你需要：

- 一个 **API Key**：有关创建 API keys 的说明，请参考 Mindee [Create & Manage API Keys documentation](https://developers.mindee.com/docs/create-api-key)。

## 使用 receipt API key <a href="#using-receipt-api-key" id="using-receipt-api-key"></a>

要配置此 credential，你需要：

- 一个 **API Key**：有关创建 API keys 的说明，请参考 Mindee [Create & Manage API Keys documentation](https://developers.mindee.com/docs/create-api-key)。
