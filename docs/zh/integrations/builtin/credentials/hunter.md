# Hunter credentials <a href="#hunter-credentials" id="hunter-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [Hunter](../app-nodes/n8n-nodes-base.hunter.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Hunter](https://www.hunter.io/) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Hunter's API documentation](https://hunter.io/api-documentation/v2)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- **API Key**：从 [dashboard](https://hunter.io/api-keys) 中的个人资料生成 API key。更多信息请参考 [Hunter API Authentication documentation](https://hunter.io/api-documentation/v2#authentication)。
