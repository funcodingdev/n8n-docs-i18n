# MessageBird credentials <a href="#messagebird-credentials" id="messagebird-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [MessageBird](../app-nodes/n8n-nodes-base.messagebird.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Bird](https://bird.com/) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [MessageBird's API documentation](https://docs.bird.com/api)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- **API Key**：要生成合适的 key，请访问 MessageBird 中的 [Access keys](https://app.bird.com/settings/access-keys) 页面。详细说明请参考 [API authorization documentation](https://docs.bird.com/api/api-access/api-authorization)。
