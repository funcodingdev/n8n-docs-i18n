## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [SolarWinds IPAM's API documentation](https://github.com/solarwinds/OrionSDK/wiki/REST)。

## 使用 Username & Password <a href="#using-username-and-password" id="using-username-and-password"></a>

要配置此 credential，你需要一个 SolarWinds IPAM 账号，以及：

- **URL**：你的 SolarWinds IPAM server 的 base URL
- **Username**：用于访问 SolarWinds IPAM 的 username
- **Password**：用于访问 SolarWinds IPAM 的 password

有关向该服务进行身份验证的更多信息，请参考 [SolarWinds IPAM's API documentation](https://github.com/solarwinds/OrionSDK/wiki/REST)。
