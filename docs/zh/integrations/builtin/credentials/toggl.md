# Toggl credentials <a href="#toggl-credentials" id="toggl-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [Toggl Trigger](../trigger-nodes/n8n-nodes-base.toggltrigger.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Toggl](https://toggl.com/) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- Basic auth

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Toggl's API documentation](https://engineering.toggl.com/docs/)。

## 使用 basic auth <a href="#using-basic-auth" id="using-basic-auth"></a>

要配置此 credential，你需要：

- 一个 **Username**：输入你的 user email address。
- 一个 **Password**：输入你的 user password。

更多信息请参考 [Authentication](https://engineering.toggl.com/docs/authentication)。
