# Vonage credentials <a href="#vonage-credentials" id="vonage-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [Vonage](../app-nodes/n8n-nodes-base.vonage.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Vonage developer](https://developer.vonage.com) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Vonage's SMS API documentation](https://developer.vonage.com/en/api/sms)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- 一个 **API Key**
- 一个 **API Secret**

从你的 [developer dashboard](https://dashboard.nexmo.com/) **user account > Settings > API Settings** 获取 **API Key** 和 **API Secret**。更多信息请参考 [Retrieve your account information](https://developer.vonage.com/en/account/guides/dashboard-management#retrieve-your-account-information)。
