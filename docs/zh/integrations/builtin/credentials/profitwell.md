# ProfitWell credentials <a href="#profitwell-credentials" id="profitwell-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [ProfitWell](../app-nodes/n8n-nodes-base.profitwell.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [ProfitWell](https://www2.profitwell.com/signup/start) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API token

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Profitwell's API documentation](https://profitwellapiv2.docs.apiary.io/)。

## 使用 API token <a href="#using-api-token" id="using-api-token"></a>

要配置此 credential，你需要：

- 一个 **API Token**：要获取 API key 或 token，请前往 **Account Settings > Integrations** 并选择 **ProfitWell API**。
