# Mocean credentials <a href="#mocean-credentials" id="mocean-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [Mocean](../app-nodes/n8n-nodes-base.mocean.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Mocean](https://moceanapi.com/) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Mocean's API documentation](https://moceanapi.com/docs/)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- 一个 **API Key**
- 一个 **API Secret**

key 和 secret 都可以在你的 Mocean [Dashboard](https://dashboard.moceanapi.com/) 中访问。更多信息请参考 [API Authentication](https://moceanapi.com/docs/#authentication)。
