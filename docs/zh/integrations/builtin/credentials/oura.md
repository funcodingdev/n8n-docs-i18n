# Oura credentials <a href="#oura-credentials" id="oura-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [Oura](../app-nodes/n8n-nodes-base.oura.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Oura](https://ouraring.com/developer) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API access token

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Oura's API documentation](https://cloud.ouraring.com/v2/docs)。

## 使用 API access token <a href="#using-api-access-token" id="using-api-access-token"></a>

要配置此 credential，你需要：

- 一个 **Personal Access Token**：要生成 personal access token，请前往 [Personal Access Tokens](https://cloud.ouraring.com/personal-access-tokens) page 并选择 **Create A New Personal Access Token**。

更多信息请参考 [How to Generate Personal Access Tokens](https://support.ouraring.com/hc/en-us/articles/4415266939155-The-Oura-API#h_01H5B94SP4P9YHG9ZKN1H69E7Z)。
