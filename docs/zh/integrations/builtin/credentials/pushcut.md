# Pushcut credentials <a href="#pushcut-credentials" id="pushcut-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [Pushcut](../app-nodes/n8n-nodes-base.pushcut.md)
- [Pushcut Trigger](../trigger-nodes/n8n-nodes-base.pushcuttrigger.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

下载 [Pushcut](https://www.pushcut.io) app。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Pushcut's Guides documentation](https://www.pushcut.io/guides)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- 一个 **API Key**：要生成 API key，请前往 **Account > Integrations > Add API Key**。更多信息请参考 [Create an API key](https://www.pushcut.io/guides/homekit-api-schedule-cancel#step_2)。
