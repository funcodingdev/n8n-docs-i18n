# Perplexity credentials <a href="#perplexity-credentials" id="perplexity-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

* [Perplexity](../app-nodes/n8n-nodes-langchain.perplexity.md)

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

* API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Perplexity's API documentation](https://docs.perplexity.ai/api-reference/)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要一个 [Perplexity account](https://www.perplexity.ai/account)，以及：

- **Perplexity API key**：你可以在 [Perplexity API getting started guide](https://docs.perplexity.ai/guides/getting-started) 中了解如何创建 Perplexity API key。

有关对该服务进行身份验证的更多信息，请参考 [Perplexity's API documentation](https://docs.perplexity.ai/)。
