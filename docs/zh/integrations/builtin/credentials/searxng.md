# SearXNG credentials <a href="#searxng-credentials" id="searxng-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

* [SearXNG Tool](../cluster-nodes/sub-nodes/n8n-nodes-langchain.toolsearxng.md)

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

* API URL

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [SearXNG's documentation](https://docs.searxng.org/index.html)。

## 使用 API URL <a href="#using-api-url" id="using-api-url"></a>

要配置此 credential，你需要有一个 n8n 可访问 URL 的 SearXNG 实例正在运行：

- **API URL**：要连接的 SearXNG 实例 URL。

有关运行该服务的更多信息，请参考 [SearXNG's Administrator documentation](https://docs.searxng.org/admin/index.html)。
