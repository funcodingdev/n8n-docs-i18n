# Malcore credentials <a href="#malcore-credentials" id="malcore-credentials"></a>

{% include "https://app.gitbook.com/s/GixZThfitWP21x2gQFpD/~/reusable/7QbEnpnpOks3Rq0SiMFb/" %}

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Malcore](https://app.malcore.io/register) 账号。

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关如何对该服务进行身份验证的更多信息，请参考 [Malcore's API documentation](https://malcore.readme.io/reference/)。

这是一个仅 credential 的 node。请参考 [Custom API operations](../custom-api-actions-for-existing-nodes.md) 了解更多。在 n8n 网站上查看[示例 workflows 和相关内容](https://n8n.io/integrations/malcore/)。


## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- **API Key**：从你的 **Account > API** 获取 API Key。

更多信息请参考 [Using the Malcore API](https://support.malcore.io/hc/en-au/articles/17711707070617-Using-the-Malcore-API)。
