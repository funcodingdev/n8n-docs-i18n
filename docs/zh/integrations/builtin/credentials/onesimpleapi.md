# One Simple API credentials <a href="#one-simple-api-credentials" id="one-simple-api-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [One Simple API](../app-nodes/n8n-nodes-base.onesimpleapi.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [One Simple API](https://onesimpleapi.com/register) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API token

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [One Simple API's documentation](https://onesimpleapi.com/docs)。

## 使用 API token <a href="#using-api-token" id="using-api-token"></a>

要配置此 credential，你需要：

- 一个 **API token**：在 [API Tokens](https://onesimpleapi.com/user/api-tokens) page 上创建新的 API token。务必为该 token 选择合适的 permissions。

你也可以通过选择 **Profile > API Tokens** 访问 API Tokens page。
