# LingvaNex credentials <a href="#lingvanex-credentials" id="lingvanex-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [LingvaNex](../app-nodes/n8n-nodes-base.lingvanex.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [LingvaNex](https://lingvanex.com) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Lingvanex's Cloud API documentation](https://docs.lingvanex.com/reference/overview)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- **API Key**：从你的 **Account** 页面生成 API key。更详细说明请参考[在哪里可以获取 authorization key？](https://docs.lingvanex.com/reference/translator-service-faq#where-can-i-get-the-authorization-key)。
