# seven credentials <a href="#seven-credentials" id="seven-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [seven](../app-nodes/n8n-nodes-base.sms77.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [seven](https://www.seven.io/en) developer account。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [seven's API documentation](https://docs.seven.io/en)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- 一个 **API key**：前往 **Account > Developer >** [**API Keys**](https://app.seven.io/developer#create-api-key) 创建 API key。更多信息请参考 [API First Steps](https://docs.seven.io/en/rest-api/first-steps)。
