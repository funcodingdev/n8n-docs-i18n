# Copper credentials <a href="#copper-credentials" id="copper-credentials"></a>

你可以使用这些 credentials 验证以下 nodes：

- [Copper](../app-nodes/n8n-nodes-base.copper.md)
- [Copper Trigger](../trigger-nodes/n8n-nodes-base.coppertrigger.md)

## 前置条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 **Professional** 或 **Business** plan 级别的 [Copper](https://www.copper.com/) 账号。

## 支持的身份验证方法 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参阅 [Copper 的 API 文档](https://developer.copper.com/)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- 一个 **API Key**：有关生成 API key 的信息，请参阅 [Copper Generating an API key 文档](https://support.copper.com/en/articles/8823347-generating-an-api-key)。
- 一个 **Email** address：使用 API key 创建者的 email address
