# Imperva WAF credentials <a href="#imperva-waf-credentials" id="imperva-waf-credentials"></a>

{% include "https://app.gitbook.com/s/GixZThfitWP21x2gQFpD/~/reusable/7QbEnpnpOks3Rq0SiMFb/" %}

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Imperva WAF](https://www.imperva.com/products/web-application-firewall-waf/) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Imperva WAF's documentation](https://docs.imperva.com/bundle/api-docs/page/api/authentication.htm)。

这是一个仅 credential 的 node。请参考 [Custom API operations](../custom-api-actions-for-existing-nodes.md) 了解更多。在 n8n 网站上查看[示例 workflows 和相关内容](https://n8n.io/integrations/imperva-waf/)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- **API ID**
- **API Key**

生成和查看 API Keys 与 IDs 的说明请参考 [Imperva WAF's API Key Management documentation](https://docs.imperva.com/bundle/cloud-application-security/page/settings/api-keys.htm)。
