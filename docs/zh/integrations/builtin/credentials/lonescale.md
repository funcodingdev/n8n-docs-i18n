# LoneScale credentials <a href="#lonescale-credentials" id="lonescale-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [LoneScale](../app-nodes/n8n-nodes-base.lonescale.md)
- [LoneScale Trigger](../trigger-nodes/n8n-nodes-base.lonescaletrigger.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [LoneScale](https://www.lonescale.com/) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [LoneScale's API documentation](https://public-api.lonescale.com/api)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- **API Key**：请参考 [LoneScale 生成 API key 文档](https://help-center.lonescale.com/en/articles/6454360-lonescale-public-api) 生成你的 key。
