## 支持的身份验证方法 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参阅 [ConvertAPI 的 API 文档](https://docs.convertapi.com/docs/getting-started)。

这是一个仅 credential 的 node。请参阅 [Custom API operations](../custom-api-actions-for-existing-nodes.md) 了解更多信息。可在 n8n 网站查看[示例 workflows 和相关内容](https://n8n.io/integrations/convertapi/)。

## 使用 API Token <a href="#using-api-token" id="using-api-token"></a>

要配置此 credential，你需要一个 [ConvertAPI](https://www.convertapi.com/a/signin) 账号，以及：

- 一个用于验证 service requests 的 [**API Token**](https://docs.convertapi.com/docs/api-tokens)。

有关向该服务进行身份验证的更多信息，请参阅 [ConvertAPI 的 API 文档](https://docs.convertapi.com/docs/authentication)。
