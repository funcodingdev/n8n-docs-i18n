# NASA credentials <a href="#nasa-credentials" id="nasa-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [NASA](../app-nodes/n8n-nodes-base.nasa.md)

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [NASA Open APIs](https://api.nasa.gov/) 的 **Browse APIs** section。

## 使用 API key <a href="#using-an-api-key" id="using-an-api-key"></a>

要配置此 credential，你需要：

- 一个 **API Key**

生成 API key：

1. 前往 [NASA Open APIs](https://api.nasa.gov/) page。
2. 完成 **Generate API Key** section 中的字段。
3. 复制 **API Key**，并将其输入到你的 n8n credential 中。
