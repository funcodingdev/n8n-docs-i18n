# Cisco Meraki credentials <a href="#cisco-meraki-credentials" id="cisco-meraki-credentials"></a>

## 前置条件 <a href="#prerequisites" id="prerequisites"></a>

- 创建一个 [Cisco DevNet developer account](https://developer.cisco.com)。
- 访问一个 [Cisco Meraki account](https://meraki.cisco.com/)。

## 身份验证方法 <a href="#authentication-methods" id="authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参阅 [Cisco Meraki 的 API 文档](https://developer.cisco.com/meraki/api-v1/introduction/)。

这是一个仅 credential 的 node。请参阅 [Custom API operations](../custom-api-actions-for-existing-nodes.md) 了解更多信息。可在 n8n 网站查看[示例 workflows 和相关内容](https://n8n.io/integrations/cisco-meraki/)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- 一个 **API Key**：有关获取 API Key 的说明，请参阅 [Cisco Meraki Obtaining your Meraki API Key 文档](https://developer.cisco.com/meraki/api-v1/authorization/#obtaining-your-meraki-api-key)。
