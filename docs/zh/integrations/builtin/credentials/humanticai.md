# Humantic AI credentials <a href="#humantic-ai-credentials" id="humantic-ai-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [Humantic AI](../app-nodes/n8n-nodes-base.humanticai.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Humantic AI](https://humantic.ai/) 账号。

你也可以在 [Humantic AI API](https://api.humantic.ai/) 页面试用 free trial API key。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Humantic AI's API documentation](https://api.humantic.ai)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- **API Key**：从 [Humantic AI API](https://api.humantic.ai/) 页面获取 API key。
