## 使用 API Token <a href="#using-api-token" id="using-api-token"></a>

要配置此 credential，你需要一个 SolarWinds Observability SaaS 账号，以及：

- **URL**：用于访问 SolarWinds Observability SaaS platform 的 URL
- **API Token**：在 SolarWinds Observability SaaS platform 的 **Settings > Api Tokens** 下找到的 API token

有关向该服务进行身份验证的更多信息，请参考 [SolarWinds Observability SaaS's API documentation](https://documentation.solarwinds.com/en/success_center/observability/content/settings/api-tokens.htm)。
