# Tapfiliate credentials <a href="#tapfiliate-credentials" id="tapfiliate-credentials"></a>

你可以使用这些 credentials 对以下 nodes 进行身份验证：

- [Tapfiliate](../app-nodes/n8n-nodes-base.tapfiliate.md)

## 前提条件 <a href="#prerequisites" id="prerequisites"></a>

创建一个 [Tapfiliate](https://tapfiliate.com/) 账号。

## 支持的身份验证方式 <a href="#supported-authentication-methods" id="supported-authentication-methods"></a>

- API key

## 相关资源 <a href="#related-resources" id="related-resources"></a>

有关该服务的更多信息，请参考 [Tapfiliate's API documentation](https://tapfiliate.com/docs/rest/)。

## 使用 API key <a href="#using-api-key" id="using-api-key"></a>

要配置此 credential，你需要：

- 一个 **API Key**：从你的 [**Profile Settings**](https://app.tapfiliate.com/a/profile/) **> API Key** 获取 API Key。

更多信息请参考 [Your API key](https://support.tapfiliate.com/en/articles/1441950-your-api-key)。
