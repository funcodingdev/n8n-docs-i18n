# Dropbox node <a href="#dropbox-node" id="dropbox-node"></a>

使用 Dropbox node 自动化 Dropbox 中的工作，并将 Dropbox 与其他应用集成。n8n 内置支持大量 Dropbox 功能，包括创建、下载、移动和复制 file 与 folder。

在本页中，你可以找到 Dropbox node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Dropbox credentials](../credentials/dropbox.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* File
    * 复制 file
    * 删除 file
    * 下载 file
    * 移动 file
    * 上传 file
* Folder
    * 复制 folder
    * 创建 folder
    * 删除 folder
    * 返回给定 folder 中的 file 和 folder
    * 移动 folder
* Search
    * Query
