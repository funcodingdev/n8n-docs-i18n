# Mocean node <a href="#mocean-node" id="mocean-node"></a>

使用 Mocean node 自动化 Mocean 中的工作，并将 Mocean 与其他应用集成。n8n 内置支持大量 Mocean 功能，包括发送短信和语音消息。

在本页中，你可以找到 Mocean node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Mocean credentials](../credentials/mocean.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* SMS
    * 发送短信/语音消息
* Voice
    * 发送短信/语音消息
