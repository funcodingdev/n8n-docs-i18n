# Snowflake node <a href="#snowflake-node" id="snowflake-node"></a>

使用 Snowflake node 自动化 Snowflake 中的工作，并将 Snowflake 与其他应用集成。n8n 内置支持大量 Snowflake 功能，包括执行 SQL query，以及向数据库中插入 row。

在本页中，你可以找到 Snowflake node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Snowflake credentials](../credentials/snowflake.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 执行 SQL query。
* 向数据库中插入 row。
* 更新数据库中的 row。
