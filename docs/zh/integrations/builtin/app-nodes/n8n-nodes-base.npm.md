# npm node <a href="#npm-node" id="npm-node"></a>

使用 npm node 自动化 npm 中的工作，并将 npm 与其他应用集成。

在本页中，你可以找到 npm node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [npm credentials](../credentials/npm.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Package
    * 获取 package metadata
    * 获取 package version
    * 搜索 package
* Distribution Tag
    * 获取所有 tag
    * 更新 tag
