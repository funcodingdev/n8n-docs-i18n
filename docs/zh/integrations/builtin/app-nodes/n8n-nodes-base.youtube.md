# YouTube node <a href="#youtube-node" id="youtube-node"></a>

使用 YouTube node 自动化 YouTube 中的工作，并将 YouTube 与其他应用集成。n8n 内置支持大量 YouTube 功能，包括检索和更新频道，以及创建和删除播放列表。

在本页中，你可以找到 YouTube node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [YouTube credentials](../credentials/google/README.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 频道
    * 检索频道
    * 检索所有频道
    * 更新频道
    * 上传频道横幅
* 播放列表
    * 创建播放列表
    * 删除播放列表
    * 获取播放列表
    * 检索所有播放列表
    * 更新播放列表
* 播放列表项目
    * 向播放列表添加项目
    * 从播放列表删除项目
    * 获取播放列表的项目
    * 检索所有播放列表项目
* 视频
    * 删除视频
    * 获取视频
    * 检索所有视频
    * 为视频评分
    * 更新视频
    * 上传视频
* 视频类别
    * 检索所有视频类别
