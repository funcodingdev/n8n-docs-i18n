# Magento 2 node <a href="#magento-2-node" id="magento-2-node"></a>

使用 Magento 2 node 自动化 Magento 2 中的工作，并将 Magento 2 与其他应用集成。n8n 内置支持大量 Magento 2 功能，包括创建、更新、删除和获取 customer、invoice、order 与 product。

在本页中，你可以找到 Magento 2 node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Magento 2 credentials](../credentials/magento2.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Customer
    * 创建新 customer
    * 删除 customer
    * 获取 customer
    * 获取所有 customer
    * 更新 customer
* Invoice
    * 创建 invoice
* Order
    * 取消 order
    * 获取 order
    * 获取所有 order
    * 发货 order
* Product
    * 创建 product
    * 删除 product
    * 获取 product
    * 获取所有 product
    * 更新 product
