# Monica CRM node <a href="#monica-crm-node" id="monica-crm-node"></a>

使用 Monica CRM node 自动化 Monica CRM 中的工作，并将 Monica CRM 与其他应用集成。n8n 内置支持大量 Monica CRM 功能，包括创建、更新、删除和获取 activity、call、contract、message、task 与 note。

在本页中，你可以找到 Monica CRM node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Monica CRM credentials](../credentials/monicacrm.md)。
{% endhint %}
