# FileMaker node <a href="#filemaker-node" id="filemaker-node"></a>

使用 FileMaker node 自动化 FileMaker 中的工作，并将 FileMaker 与其他应用集成。n8n 内置支持大量 FileMaker 功能，包括创建、查找、获取、编辑和复制 file。

在本页中，你可以找到 FileMaker node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [FileMaker credentials](../credentials/filemaker.md)。
{% endhint %}

- 删除 Record
