# Microsoft To Do node <a href="#microsoft-to-do-node" id="microsoft-to-do-node"></a>

使用 Microsoft To Do node 自动化 Microsoft To Do 中的工作，并将 Microsoft To Do 与其他应用集成。n8n 内置支持大量 Microsoft To Do 功能，包括创建、更新、删除和获取 linked resource、list 与 task。

在本页中，你可以找到 Microsoft To Do node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Microsoft credentials](../credentials/microsoft.md)。
{% endhint %}

{% hint style="info" %}
**政府云支持**

如果你使用 government cloud tenant（US Government、US Government DOD 或 China），请确保在 Microsoft credentials 配置中选择合适的 **Microsoft Graph API Base URL**。
{% endhint %}
