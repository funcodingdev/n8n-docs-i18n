# HighLevel node <a href="#highlevel-node" id="highlevel-node"></a>

使用 HighLevel node 自动化 HighLevel 中的工作，并将 HighLevel 与其他应用集成。n8n 内置支持大量 HighLevel 功能，包括创建、更新、删除和获取 contact、opportunity 与 task，以及预订 appointment 和获取 calendar 中的空闲时段。

在本页中，你可以找到 HighLevel node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [HighLevel credentials](../credentials/highlevel.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Contact
    * 创建或更新
    * Delete
    * Get
    * Get many
    * Update
* Opportunity
    * Create
    * Delete
    * Get
    * Get many
    * Update
* Task
    * Create
    * Delete
    * Get
    * Get many
    * Update
* Calendar
    * 预订 appointment
    * 获取空闲 slot
