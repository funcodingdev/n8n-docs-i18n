# uProc node <a href="#uproc-node" id="uproc-node"></a>

使用 uProc node 自动化 uProc 中的工作，并将 uProc 与其他应用集成。n8n 内置支持大量 uProc 功能，包括获取高级人声音频文件、通信数据、公司信息、财务信息和产品信息。

在本页中，你可以找到 uProc node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [uProc credentials](../credentials/uproc.md)。
{% endhint %}
