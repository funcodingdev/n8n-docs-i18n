# Freshworks CRM node <a href="#freshworks-crm-node" id="freshworks-crm-node"></a>

使用 Freshworks CRM node 自动化 Freshworks CRM 中的工作，并将 Freshworks CRM 与其他应用集成。n8n 内置支持大量 Freshworks CRM 功能，包括创建、更新、删除和检索 account、appointment、contact、deal、note、sales activity 等。

在本页中，你可以找到 Freshworks CRM node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Freshworks CRM credentials](../credentials/freshworkscrm.md)。
{% endhint %}
