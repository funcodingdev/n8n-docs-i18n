# seven node <a href="#seven-node" id="seven-node"></a>

使用 seven node 自动化 seven 中的工作，并将 seven 与其他应用集成。n8n 内置支持大量 seven 功能，包括发送 SMS 和将文本转换为语音。

在本页中，你可以找到 seven node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [seven credentials](../credentials/sms77.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* SMS
    * 发送 SMS
* Voice Call
    * 将文本转换为语音，并呼叫给定号码
