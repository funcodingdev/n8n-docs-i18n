# Flow node <a href="#flow-node" id="flow-node"></a>

使用 Flow node 自动化 Flow 中的工作，并将 Flow 与其他应用集成。n8n 内置支持大量 Flow 功能，包括创建、更新和获取 task。

在本页中，你可以找到 Flow node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Flow credentials](../credentials/flow.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Task
    * 创建新 task
    * 更新 task
    * 获取 task
    * 获取所有 task
