# SeaTable node <a href="#seatable-node" id="seatable-node"></a>

使用 SeaTable node 自动化 SeaTable 中的工作，并将 SeaTable 与其他应用集成。n8n 内置支持大量 SeaTable 功能，包括创建、更新、删除和获取 row。

在本页中，你可以找到 SeaTable node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [SeaTable credentials](../credentials/seatable.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Row
    * 创建
    * 删除
    * 获取
    * 获取全部
    * 更新
