# Mailjet node <a href="#mailjet-node" id="mailjet-node"></a>

使用 Mailjet node 自动化 Mailjet 中的工作，并将 Mailjet 与其他应用集成。n8n 内置支持大量 Mailjet 功能，包括发送 email 和 SMS。

在本页中，你可以找到 Mailjet node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Mailjet credentials](../credentials/mailjet.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Email
    * 发送 email
    * 发送 email template
* SMS
    * 发送 SMS
