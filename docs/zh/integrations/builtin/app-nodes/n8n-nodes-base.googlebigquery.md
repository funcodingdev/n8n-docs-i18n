# Google BigQuery node <a href="#google-bigquery-node" id="google-bigquery-node"></a>

使用 Google BigQuery node 自动化 Google BigQuery 中的工作，并将 Google BigQuery 与其他应用集成。n8n 内置支持大量 Google BigQuery 功能，包括创建和检索 record。

在本页中，你可以找到 Google BigQuery node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Google BigQuery credentials](../credentials/google/README.md)。
{% endhint %}
