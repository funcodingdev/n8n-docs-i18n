# MongoDB node <a href="#mongodb-node" id="mongodb-node"></a>

使用 MongoDB node 自动化 MongoDB 中的工作，并将 MongoDB 与其他应用集成。n8n 内置支持大量 MongoDB 功能，包括聚合、更新、查找、删除和获取 document，以及创建、更新、列出和删除 search index。此 node 中的所有操作都使用 [MongoDB Node driver](https://www.mongodb.com/docs/drivers/node/current/)。

在本页中，你可以找到 MongoDB node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [MongoDB credentials](../credentials/mongodb.md)。
{% endhint %}
