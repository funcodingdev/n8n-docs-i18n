# Brevo node <a href="#brevo-node" id="brevo-node"></a>

使用 Brevo node 自动化 Brevo 中的工作，并将 Brevo 与其他应用集成。n8n 内置支持大量 Brevo 功能，包括创建、更新、删除和获取 contact、attribute，以及发送 email。

在本页中，你可以找到 Brevo node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Brevo credentials](../credentials/brevo.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Contact
    * Create
    * Create or Update
    * Delete
    * Get
    * Get All
    * Update
* Contact Attribute
    * Create
    * Delete
    * Get All
    * Update
* Email
    * Send
    * Send Template
* Sender
    * Create
    * Delete
    * Get All
