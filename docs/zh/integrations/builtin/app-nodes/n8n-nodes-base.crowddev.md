# crowd.dev node <a href="#crowddev-node" id="crowddev-node"></a>

使用 crowd.dev node 自动化 crowd.dev 中的工作，并将 crowd.dev 与其他应用集成。n8n 内置支持大量 crowd.dev 功能，包括创建、更新和删除 member、note、organization 与 task。

在本页中，你可以找到 crowd.dev node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/crowddev.md)找到此 node 的身份验证信息。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Activity
    * 使用 member 创建或更新
    * Create
* Automation
    * Create
    * Destroy
    * Find
    * List
    * Update
* Member
    * 创建或更新
    * Delete
    * Find
    * Update
* Note
    * Create
    * Delete
    * Find
    * Update
* Organization
    * Create
    * Delete
    * Find
    * Update
* Task
    * Create
    * Delete
    * Find
    * Update
