# AWS Certificate Manager node <a href="#aws-certificate-manager-node" id="aws-certificate-manager-node"></a>

使用 AWS Certificate Manager node 自动化 AWS Certificate Manager 中的工作，并将 AWS Certificate Manager 与其他应用集成。n8n 内置支持大量 AWS Certificate Manager 功能，包括创建、删除、获取和续订 SSL certificate。

在本页中，你可以找到 AWS Certificate Manager node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [AWS Certificate Manager credentials](../credentials/aws.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Certificate
	* Delete
	* Get
	* Get Many
	* Get Metadata
	* Renew
