# Elastic Security node <a href="#elastic-security-node" id="elastic-security-node"></a>

使用 Elastic Security node 自动化 Elastic Security 中的工作，并将 Elastic Security 与其他应用集成。n8n 内置支持大量 Elastic Security 功能，包括创建、更新、删除、检索和获取 case。

在本页中，你可以找到 Elastic Security node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Elastic Security credentials](../credentials/elasticsecurity.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Case
    * 创建 case
    * 删除 case
    * 获取 case
    * 检索所有 case
    * 检索所有 case 活动摘要
    * 更新 case
* Case Comment
    * 向 case 添加 comment
    * 获取 case comment
    * 检索所有 case comment
    * 从 case 中移除 comment
    * 更新 case 中的 comment
* Case Tag
    * 向 case 添加 tag
    * 从 case 中移除 tag
* Connector
    * 创建 connector
