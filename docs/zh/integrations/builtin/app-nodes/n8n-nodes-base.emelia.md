# Emelia node <a href="#emelia-node" id="emelia-node"></a>

使用 Emelia node 自动化 Emelia 中的工作，并将 Emelia 与其他应用集成。n8n 内置支持大量 Emelia 功能，包括创建 campaign，以及将 contact 添加到 list。

在本页中，你可以找到 Emelia node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Emelia credentials](../credentials/emelia.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Campaign
    * 添加 Contact
    * Create
    * Get
    * Get All
    * Pause
    * Start
* Contact List
    * Add
    * Get All
