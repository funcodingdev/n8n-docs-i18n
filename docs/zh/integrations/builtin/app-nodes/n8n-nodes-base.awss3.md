# AWS S3 node <a href="#aws-s3-node" id="aws-s3-node"></a>

使用 AWS S3 node 自动化 AWS S3 中的工作，并将 AWS S3 与其他应用集成。n8n 内置支持大量 AWS S3 功能，包括创建和删除 bucket、复制和下载文件，以及获取 folder。

在本页中，你可以找到 AWS S3 node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [AWS credentials](../credentials/aws.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Bucket
    * 创建 bucket
    * 删除 bucket
    * 获取所有 bucket
    * 在 bucket 内搜索
* File
    * 复制文件
    * 删除文件
    * 下载文件
    * 获取所有文件
    * 上传文件
* Folder
    * 创建 folder
    * 删除 folder
    * 获取所有 folder
