# ERPNext node <a href="#erpnext-node" id="erpnext-node"></a>

使用 ERPNext node 自动化 ERPNext 中的工作，并将 ERPNext 与其他应用集成。n8n 内置支持大量 ERPNext 功能，包括创建、更新、检索和删除 document。

在本页中，你可以找到 ERPNext node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [ERPNext credentials](../credentials/erpnext.md)。
{% endhint %}
