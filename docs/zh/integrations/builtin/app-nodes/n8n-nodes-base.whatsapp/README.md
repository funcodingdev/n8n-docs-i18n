# WhatsApp Business Cloud node <a href="#whatsapp-business-cloud-node" id="whatsapp-business-cloud-node"></a>

使用 WhatsApp Business Cloud node 自动化 WhatsApp Business 中的工作，并将 WhatsApp Business 与其他应用集成。n8n 内置支持大量 WhatsApp Business 功能，包括发送消息，以及上传、下载和删除媒体。

在本页中，你可以找到 WhatsApp Business Cloud node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials 凭据**

有关身份验证设置指南，请参阅 [WhatsApp Business Cloud credentials](../../credentials/whatsapp.md)。
{% endhint %}
