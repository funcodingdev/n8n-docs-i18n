# Medium node <a href="#medium-node" id="medium-node"></a>

使用 Medium node 自动化 Medium 中的工作，并将 Medium 与其他应用集成。n8n 内置支持大量 Medium 功能，包括创建 post 和获取 publication。

在本页中，你可以找到 Medium node 支持的操作列表，以及更多资源链接。

{% hint style="warning" %}
**Medium API 不再受支持**

Medium 已停止支持 Medium API。Medium node 仍会显示在 n8n 中，但你将无法配置新的 API key 用于身份验证。

有关设置现有 API key 的指导，请参阅 [Medium credentials](../credentials/medium.md)。
{% endhint %}
