# Zammad node <a href="#zammad-node" id="zammad-node"></a>

使用 Zammad node 自动化 Zammad 中的工作，并将 Zammad 与其他应用集成。n8n 内置支持大量 Zammad 功能，包括创建、检索和删除群组与组织。

在本页中，你可以找到 Zammad node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Zammad credentials](../credentials/zammad.md)。
{% endhint %}

* 组织

    * 删除
    * 获取
    * 获取多个
    * 更新
* 工单
    * 创建
    * 删除
    * 获取
    * 获取多个
    * 更新
* 用户
    * 创建
    * 删除
    * 获取
