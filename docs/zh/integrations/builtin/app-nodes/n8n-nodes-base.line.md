# Line node <a href="#line-node" id="line-node"></a>

{% hint style="warning" %}
**已弃用：服务终止**

LINE Notify 将于 2025 年 4 月 1 日停止服务，此日期之后该 node 将不再工作。有关更多信息，请查看 LINE Notify 的[服务终止公告](https://notify-bot.line.me/closing-announce)。
{% endhint %}

使用 Line node 自动化 Line 中的工作，并将 Line 与其他应用集成。n8n 内置支持大量 Line 功能，包括发送 notification。

在本页中，你可以找到 Line node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Line credentials](../credentials/line.md)。
{% endhint %}
