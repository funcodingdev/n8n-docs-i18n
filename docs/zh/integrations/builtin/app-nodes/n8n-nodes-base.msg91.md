# MSG91 node <a href="#msg91-node" id="msg91-node"></a>

使用 MSG91 node 自动化 MSG91 中的工作，并将 MSG91 与其他应用集成。n8n 支持通过 MSG91 发送短信。

在本页中，你可以找到 MSG91 node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [MSG91 credentials](../credentials/msg91.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* SMS
    * 发送短信

[浏览 MSG91 node 文档集成模板](https://n8n.io/integrations/msg91)或[搜索所有模板](https://n8n.io/workflows/)

## 查找你的 Sender ID <a href="#find-your-sender-id" id="find-your-sender-id"></a>

1. 登录你的 MSG91 dashboard。
2. 在左侧面板中选择 **Sender Id**。
3. 如果还没有 Sender ID，选择 **Add Sender Id +**，填写详细信息，然后选择 **Save Sender Id**。
