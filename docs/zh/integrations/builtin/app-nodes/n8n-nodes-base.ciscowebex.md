# Webex by Cisco node <a href="#webex-by-cisco-node" id="webex-by-cisco-node"></a>

使用 Webex by Cisco node 自动化 Webex 中的工作，并将 Webex 与其他应用集成。n8n 内置支持大量 Webex 功能，包括创建、获取、更新和删除 meeting 与 message。

在本页中，你可以找到 Webex node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Webex credentials](../credentials/ciscowebex.md)。
{% endhint %}
{% hint style="info" %}
**示例和模板**

有关帮助你上手的使用示例和模板，请查看 n8n 的 [Webex integrations](https://n8n.io/integrations/webex-by-cisco/) 列表。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Meeting
    * Create
    * Delete
    * Get
    * Get All
    * Update
* Message
    * Create
    * Delete
    * Get
    * Get All
    * Update
