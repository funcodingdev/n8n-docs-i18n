# PhantomBuster node <a href="#phantombuster-node" id="phantombuster-node"></a>

使用 PhantomBuster node 自动化 PhantomBuster 中的工作，并将 PhantomBuster 与其他应用集成。n8n 内置支持大量 PhantomBuster 功能，包括添加、删除和获取 agent。

在本页中，你可以找到 PhantomBuster node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [PhantomBuster credentials](../credentials/phantombuster.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Agent
    * 按 ID 删除 agent。
    * 按 ID 获取 agent。
    * 获取当前用户 organization 的所有 agent。
    * 获取 agent 最近一个 container 的输出。
    * 将 agent 添加到 launch queue。
