# Keap node <a href="#keap-node" id="keap-node"></a>

使用 Keap node 自动化 Keap 中的工作，并将 Keap 与其他应用集成。n8n 内置支持大量 Keap 功能，包括创建、更新、删除和获取 company、product、ecommerce order、email 与 file。

在本页中，你可以找到 Keap node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Keap credentials](../credentials/keap.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Company
    * 创建 company
    * 检索所有 company
* Contact
    * 创建/更新 contact
    * 删除 contact
    * 检索 contact
    * 检索所有 contact
* Contact Note
    * 创建 note
    * 删除 note
    * 获取 note
    * 检索所有 note
    * 更新 note
* Contact Tag
    * 向 contact 添加 tag 列表
    * 删除 contact 的 tag
    * 检索所有 contact tag
* Ecommerce Order
    * 创建 ecommerce order
    * 获取 ecommerce order
    * 删除 ecommerce order
    * 检索所有 ecommerce order
* Ecommerce Product
    * 创建 ecommerce product
    * 删除 ecommerce product
    * 获取 ecommerce product
    * 检索所有 ecommerce product
* Email
    * 创建一条发送给 contact 的 email 记录
    * 检索所有已发送 email
    * 发送 Email
* File
    * 删除 file
    * 检索所有 file
    * 上传 file
