# Strapi node <a href="#strapi-node" id="strapi-node"></a>

使用 Strapi node 自动化 Strapi 中的工作，并将 Strapi 与其他应用集成。n8n 内置支持大量 Strapi 功能，包括创建和删除 entry。

在本页中，你可以找到 Strapi node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Strapi credentials](../credentials/strapi.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Entry
    * 创建
    * 删除
    * 获取
    * 获取多个
    * 更新
