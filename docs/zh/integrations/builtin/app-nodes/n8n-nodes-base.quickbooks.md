# QuickBooks Online node <a href="#quickbooks-online-node" id="quickbooks-online-node"></a>

使用 QuickBooks node 自动化 QuickBooks 中的工作，并将 QuickBooks 与其他应用集成。n8n 内置支持大量 QuickBooks 功能，包括创建、更新、删除和获取 bill、customer、employee、estimate 与 invoice。

在本页中，你可以找到 QuickBooks node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [QuickBooks credentials](../credentials/quickbooks.md)。
{% endhint %}
