# Google Cloud Firestore node <a href="#google-cloud-firestore-node" id="google-cloud-firestore-node"></a>

使用 Google Cloud Firestore node 自动化 Google Cloud Firestore 中的工作，并将 Google Cloud Firestore 与其他应用集成。n8n 内置支持大量 Google Cloud Firestore 功能，包括创建、删除和获取 document。

在本页中，你可以找到 Google Cloud Firestore node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Google credentials](../credentials/google/README.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Document
    * 创建 document
    * 创建/更新 document
    * 删除 document
    * 获取 document
    * 获取 collection 中的所有 document
    * 针对你的 document 运行 query
* Collection
    * 获取所有 root collection
