# Beeminder node <a href="#beeminder-node" id="beeminder-node"></a>

使用 Beeminder node 自动化 Beeminder 中的工作，并将 Beeminder 与其他应用集成。n8n 内置支持大量 Beeminder 功能，包括创建、删除和更新 data point。

在本页中，你可以找到 Beeminder node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Beeminder credentials](../credentials/beeminder.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

**data point**
- 为 goal 创建 data point
- 删除 data point
- 获取 goal 的所有 data point
- 更新 data point
