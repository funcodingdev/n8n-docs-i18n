# ProfitWell node <a href="#profitwell-node" id="profitwell-node"></a>

使用 ProfitWell node 自动化 ProfitWell 中的工作，并将 ProfitWell 与其他应用集成。n8n 支持从 ProfitWell 获取你公司的 account setting，并检索财务 metric。

在本页中，你可以找到 ProfitWell node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [ProfitWell credentials](../credentials/profitwell.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Company
    * 获取你公司的 ProfitWell account setting
* Metric
    * 检索按天拆分的财务 metric，时间范围为当月或上一周期
