# Kafka node <a href="#kafka-node" id="kafka-node"></a>

使用 Kafka node 自动化 Kafka 中的工作，并将 Kafka 与其他应用集成。n8n 内置支持大量 Kafka 功能，包括发送 message。

在本页中，你可以找到 Kafka node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Kafka credentials](../credentials/kafka.md)。
{% endhint %}

{% hint style="info" %}
**Schema Registry**

要使用已认证的 Confluent Schema Registry（例如 Confluent Cloud）编码 message，请在 node 中启用 **Use Schema Registry**，并添加 [Schema Registry credential](/integrations/builtin/credentials/schemaregistry.md)。
{% endhint %}
