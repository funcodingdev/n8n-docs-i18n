# APITemplate.io node <a href="#apitemplateio-node" id="apitemplateio-node"></a>

使用 APITemplate.io node 自动化 APITemplate.io 中的工作，并将 APITemplate.io 与其他应用集成。n8n 内置支持大量 APITemplate.io 功能，包括获取和创建 account、image 与 PDF。

在本页中，你可以找到 APITemplate.io node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [APITemplate.io credentials](../credentials/apitemplateio.md)。
{% endhint %}
