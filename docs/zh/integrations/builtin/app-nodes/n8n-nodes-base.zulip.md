# Zulip node <a href="#zulip-node" id="zulip-node"></a>

使用 Zulip node 自动化 Zulip 中的工作，并将 Zulip 与其他应用集成。n8n 内置支持大量 Zulip 功能，包括创建、删除和获取用户与频道，以及发送消息。

在本页中，你可以找到 Zulip node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Zulip credentials](../credentials/zulip.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 消息
    * 删除消息
    * 获取消息
    * 发送私信
    * 向 stream 发送消息
    * 更新消息
    * 上传文件
* 频道
    * 创建频道
    * 删除频道
    * 获取所有频道
    * 获取已订阅的频道
    * 更新频道
* 用户
    * 创建用户
    * 停用用户
    * 获取用户
    * 获取所有用户
    * 更新用户
