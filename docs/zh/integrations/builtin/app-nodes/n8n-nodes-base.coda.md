# Coda node <a href="#coda-node" id="coda-node"></a>

使用 Coda node 自动化 Coda 中的工作，并将 Coda 与其他应用集成。n8n 内置支持大量 Coda 功能，包括创建、获取和删除 control、formula、table 与 view。

在本页中，你可以找到 Coda node 支持的操作列表，以及更多资源链接。

## Credentials <a href="#credentials" id="credentials"></a>

有关设置身份验证的指导，请参阅 [Coda credentials](../credentials/coda.md)。

## 操作 <a href="#operations" id="operations"></a>

* Control
    * 获取 control
    * 获取所有 control
* Formula
    * 获取 formula
    * 获取所有 formula
* Table
    * 创建/插入 row
    * 删除一个或多个 row
    * 获取所有 column
    * 获取所有 row
    * 获取 column
    * 获取 row
    * 按下 button
* View
    * 删除 view row
    * 获取 view
    * 获取所有 view
    * 获取所有 view column
    * 获取所有 view row
    * 更新 row
    * 按下 view button
