# Freshdesk node <a href="#freshdesk-node" id="freshdesk-node"></a>

使用 Freshdesk node 自动化 Freshdesk 中的工作，并将 Freshdesk 与其他应用集成。n8n 内置支持大量 Freshdesk 功能，包括创建、更新、删除和获取 contact 与 ticket。

在本页中，你可以找到 Freshdesk node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Freshdesk credentials](../credentials/freshdesk.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Contact
    * 创建新 contact
    * 删除 contact
    * 获取 contact
    * 获取所有 contact
    * 更新 contact
* Ticket
    * 创建新 ticket
    * 删除 ticket
    * 获取 ticket
    * 获取所有 ticket
    * 更新 ticket
