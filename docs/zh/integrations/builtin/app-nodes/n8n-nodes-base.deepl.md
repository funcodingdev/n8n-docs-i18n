# DeepL node <a href="#deepl-node" id="deepl-node"></a>

使用 DeepL node 自动化 DeepL 中的工作，并将 DeepL 与其他应用集成。n8n 内置支持大量 DeepL 功能，包括翻译语言。

在本页中，你可以找到 DeepL node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [DeepL credentials](../credentials/deepl.md)。
{% endhint %}
