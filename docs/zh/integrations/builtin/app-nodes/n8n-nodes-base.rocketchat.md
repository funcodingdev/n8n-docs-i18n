# Rocket.Chat node <a href="#rocketchat-node" id="rocketchat-node"></a>

使用 Rocket.Chat node 自动化 Rocket.Chat 中的工作，并将 Rocket.Chat 与其他应用集成。n8n 支持通过 Rocket.Chat 向 channel 发帖和发送 direct message。

在本页中，你可以找到 Rocket.Chat node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Rocket.Chat credentials](../credentials/rocketchat.md)。
{% endhint %}
