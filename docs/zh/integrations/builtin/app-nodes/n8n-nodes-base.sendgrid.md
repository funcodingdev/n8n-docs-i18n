# SendGrid node <a href="#sendgrid-node" id="sendgrid-node"></a>

使用 SendGrid node 自动化 SendGrid 中的工作，并将 SendGrid 与其他应用集成。n8n 内置支持大量 SendGrid 功能，包括创建、更新、删除和获取 contact 与 list，以及发送 email。

在本页中，你可以找到 SendGrid node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [SendGrid credentials](../credentials/sendgrid.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Contact
    * 创建/更新 contact
    * 删除 contact
    * 按 ID 获取 contact
    * 获取所有 contact
* List
    * 创建 list
    * 删除 list
    * 获取 list
    * 获取所有 list
    * 更新 list
* Mail
    * 发送 email。
