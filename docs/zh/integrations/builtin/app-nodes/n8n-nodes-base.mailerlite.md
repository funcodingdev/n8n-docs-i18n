# MailerLite node <a href="#mailerlite-node" id="mailerlite-node"></a>

使用 MailerLite node 自动化 MailerLite 中的工作，并将 MailerLite 与其他应用集成。n8n 内置支持大量 MailerLite 功能，包括创建、更新、删除和获取 subscriber。

在本页中，你可以找到 MailerLite node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [MailerLite credentials](../credentials/mailerlite.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Subscriber
    * 创建新 subscriber
    * 获取 subscriber
    * 获取所有 subscriber
    * 更新 subscriber
