# Pipedrive node <a href="#pipedrive-node" id="pipedrive-node"></a>

使用 Pipedrive node 自动化 Pipedrive 中的工作，并将 Pipedrive 与其他应用集成。n8n 内置支持大量 Pipedrive 功能，包括创建、更新、删除和获取 activity、file、note、organization 与 lead。

在本页中，你可以找到 Pipedrive node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Pipedrive credentials](../credentials/pipedrive.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Activity
    * 创建 activity
    * 删除 activity
    * 获取 activity 数据
    * 获取所有 activity 的数据
    * 更新 activity
* Deal
    * 创建 deal
    * 删除 deal
    * 复制 deal
    * 获取 deal 数据
    * 获取所有 deal 的数据
    * 搜索 deal
    * 更新 deal
* Deal Activity
    * 获取 deal 的所有 activity
* Deal Product
    * 向 deal 添加 product
    * 获取 deal 中的所有 product
    * 从 deal 中移除 product
    * 更新 deal 中的 product
* File
    * 创建 file
    * 删除 file
    * 下载 file
    * 获取 file 数据
* Lead
    * 创建 lead
    * 删除 lead
    * 获取 lead 数据
    * 获取所有 lead 的数据
    * 更新 lead
* Note
    * 创建 note
    * 删除 note
    * 获取 note 数据
    * 获取所有 note 的数据
    * 更新 note
* Organization
    * 创建 organization
    * 删除 organization
    * 获取 organization 数据
    * 获取所有 organization 的数据
    * 更新 organization
    * 搜索 organization
* Person
    * 创建 person
    * 删除 person
    * 获取 person 数据
    * 获取所有 person 的数据
    * 搜索所有 person
    * 更新 person
* Product
    * 获取所有 product 的数据
