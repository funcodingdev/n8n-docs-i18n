# Ghost node <a href="#ghost-node" id="ghost-node"></a>

使用 Ghost node 自动化 Ghost 中的工作，并将 Ghost 与其他应用集成。n8n 内置支持大量 Ghost 功能，包括通过 Admin API 和 Content API 创建、更新、删除和获取 post。

在本页中，你可以找到 Ghost node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Ghost credentials](../credentials/ghost.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

### Admin API <a href="#admin-api" id="admin-api"></a>

* **Post**
    * 创建 post
    * 删除 post
    * 获取 post
    * 获取所有 post
    * 更新 post

### Content API <a href="#content-api" id="content-api"></a>

* **Post**
    * 获取 post
    * 获取所有 post
