# Webflow node <a href="#webflow-node" id="webflow-node"></a>

使用 Webflow node 自动化 Webflow 中的工作，并将 Webflow 与其他应用集成。n8n 内置支持大量 Webflow 功能，包括创建、更新、删除和获取 item。

在本页中，你可以找到 Webflow node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Webflow credentials](../credentials/webflow.md)。
{% endhint %}
