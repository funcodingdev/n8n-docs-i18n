# Agile CRM node <a href="#agile-crm-node" id="agile-crm-node"></a>

使用 Agile CRM node 自动化 Agile CRM 中的工作，并将 Agile CRM 与其他应用集成。n8n 内置支持大量 Agile CRM 功能，包括创建、获取、更新和删除 company、contact 与 deal。

在本页中，你可以找到 Agile CRM node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Agile CRM credentials](../credentials/agilecrm.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Company
    * Create a new company
    * Delete a company
    * Get a company
    * Get all companies
    * Update company properties
* Contact
    * Create a new contact
    * Delete a contact
    * Get a contact
    * Get all contacts
    * Update contact properties
* Deal
    * Create a new deal
    * Delete a deal
    * Get a deal
    * Get all deals
    * Update deal properties
