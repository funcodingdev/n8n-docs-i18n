# Adalo node <a href="#adalo-node" id="adalo-node"></a>

使用 Adalo node 自动化 Adalo 中的工作，并将 Adalo 与其他应用集成。n8n 内置支持大量 Adalo 功能，包括创建、获取、更新和删除 database、record 与 collection。

在本页中，你可以找到 Adalo node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Adalo credentials](../credentials/adalo.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Collection
	* Create
	* Delete
	* Get
	* Get Many
	* Update
