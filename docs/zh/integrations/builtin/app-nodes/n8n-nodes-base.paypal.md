# PayPal node <a href="#paypal-node" id="paypal-node"></a>

使用 PayPal node 自动化 PayPal 中的工作，并将 PayPal 与其他应用集成。n8n 内置支持大量 PayPal 功能，包括创建 batch payout 和取消未领取的 payout item。

在本页中，你可以找到 PayPal node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [PayPal credentials](../credentials/paypal.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Payout
    * 创建 batch payout
    * 显示 batch payout 详细信息
* Payout Item
    * 取消未领取的 payout item
    * 显示 payout item 详细信息
