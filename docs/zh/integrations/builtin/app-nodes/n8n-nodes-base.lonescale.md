# LoneScale node <a href="#lonescale-node" id="lonescale-node"></a>

使用 LoneScale node 自动化 LoneScale 中的工作，并将 LoneScale 与其他应用集成。n8n 内置支持在 LoneScale 中管理 List 和 Item。

在本页中，你可以找到 LoneScale node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/lonescale.md)找到此 node 的身份验证信息。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* List
    * Create
* Item
    * Create
