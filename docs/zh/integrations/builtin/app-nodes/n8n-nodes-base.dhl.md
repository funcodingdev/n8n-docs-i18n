# DHL node <a href="#dhl-node" id="dhl-node"></a>

使用 DHL node 自动化 DHL 中的工作，并将 DHL 与其他应用集成。n8n 内置支持大量 DHL 功能，包括跟踪 shipment。

在本页中，你可以找到 DHL node 支持的操作列表，以及更多资源链接。
