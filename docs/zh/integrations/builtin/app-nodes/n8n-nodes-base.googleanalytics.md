# Google Analytics node <a href="#google-analytics-node" id="google-analytics-node"></a>

使用 Google Analytics node 自动化 Google Analytics 中的工作，并将 Google Analytics 与其他应用集成。n8n 内置支持大量 Google Analytics 功能，包括返回 report 和 user activity。

在本页中，你可以找到 Google Analytics node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Google Analytics credentials](../credentials/google/README.md)。
{% endhint %}
