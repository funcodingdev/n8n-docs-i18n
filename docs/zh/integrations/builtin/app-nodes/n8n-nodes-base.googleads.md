# Google Ads node <a href="#google-ads-node" id="google-ads-node"></a>

使用 Google Ads node 自动化 Google Ads 中的工作，并将 Google Ads 与其他应用集成。n8n 内置支持大量 Google Ads 功能，包括获取 campaign。

在本页中，你可以找到 Google Ads node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Google Ads credentials](../credentials/google/README.md)。
{% endhint %}
