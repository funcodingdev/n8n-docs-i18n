# NocoDB node <a href="#nocodb-node" id="nocodb-node"></a>

使用 NocoDB node 自动化 NocoDB 中的工作，并将 NocoDB 与其他应用集成。n8n 内置支持大量 NocoDB 功能，包括创建、更新、删除和检索 row。

在本页中，你可以找到 NocoDB node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [NocoDB credentials](../credentials/nocodb.md)。
{% endhint %}
