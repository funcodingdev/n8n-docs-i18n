# Affinity node <a href="#affinity-node" id="affinity-node"></a>

使用 Affinity node 自动化 Affinity 中的工作，并将 Affinity 与其他应用集成。n8n 内置支持大量 Affinity 功能，包括创建、获取、更新和删除 list、entry、organization 与 person。

在本页中，你可以找到 Affinity node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Affinity credentials](../credentials/affinity.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* List
    * Get a list
    * Get all lists
* List Entry
    * Create a list entry
    * Delete a list entry
    * Get a list entry
    * Get all list entries
* Organization
    * Create an organization
    * Delete an organization
    * Get an organization
    * Get all organizations
    * Update an organization
* Person
    * Create a person
    * Delete a person
    * Get a person
    * Get all persons
    * Update a person
