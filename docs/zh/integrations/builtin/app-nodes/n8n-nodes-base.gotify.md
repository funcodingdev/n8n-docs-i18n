# Gotify node <a href="#gotify-node" id="gotify-node"></a>

使用 Gotify node 自动化 Gotify 中的工作，并将 Gotify 与其他应用集成。n8n 内置支持大量 Gotify 功能，包括创建、删除和获取 message。

在本页中，你可以找到 Gotify node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Gotify credentials](../credentials/gotify.md)。
{% endhint %}
