# Invoice Ninja node <a href="#invoice-ninja-node" id="invoice-ninja-node"></a>

使用 Invoice Ninja node 自动化 Invoice Ninja 中的工作，并将 Invoice Ninja 与其他应用集成。n8n 内置支持大量 Invoice Ninja 功能，包括创建、更新、删除和获取 client、expense、invoice、payment 与 quote。

在本页中，你可以找到 Invoice Ninja node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Invoice Ninja credentials](../credentials/invoiceninja.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Client
    * 创建新 client
    * 删除 client
    * 获取 client 数据
    * 获取所有 client 的数据
* Expense
    * 创建新 expense
    * 删除 expense
    * 获取 expense 数据
    * 获取所有 expense 的数据
* Invoice
    * 创建新 invoice
    * 删除 invoice
    * 发送 invoice 邮件
    * 获取 invoice 数据
    * 获取所有 invoice 的数据
* Payment
    * 创建新 payment
    * 删除 payment
    * 获取 payment 数据
    * 获取所有 payment 的数据
* Quote
    * 创建新 quote
    * 删除 quote
    * 发送 quote 邮件
    * 获取 quote 数据
    * 获取所有 quote 的数据
* Task
    * 创建新 task
    * 删除 task
    * 获取 task 数据
    * 获取所有 task 的数据
