# Intercom node <a href="#intercom-node" id="intercom-node"></a>

使用 Intercom node 自动化 Intercom 中的工作，并将 Intercom 与其他应用集成。n8n 内置支持大量 Intercom 功能，包括创建、更新、删除和获取 company、lead 与 user。

在本页中，你可以找到 Intercom node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Intercom credentials](../credentials/intercom.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Company
    * 创建新 company
    * 获取 company 数据
    * 获取所有 company 的数据
    * 更新 company
    * 列出 company 的 user
* Lead
    * 创建新 lead
    * 删除 lead
    * 获取 lead 数据
    * 获取所有 lead 的数据
    * 更新新 lead
* User
    * 创建新 user
    * 删除 user
    * 获取 user 数据
    * 获取所有 user 的数据
    * 更新 user
