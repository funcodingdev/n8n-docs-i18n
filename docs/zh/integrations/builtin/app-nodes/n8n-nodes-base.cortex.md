# Cortex node <a href="#cortex-node" id="cortex-node"></a>

使用 Cortex node 自动化 Cortex 中的工作，并将 Cortex 与其他应用集成。n8n 内置支持大量 Cortex 功能，包括执行 analyzer 和 responder，以及获取 job 详情。

在本页中，你可以找到 Cortex node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Cortex credentials](../credentials/cortex.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Analyzer
    * 执行 Analyzer
* Job
    * 获取 job 详情
    * 获取 job report
* Responder
    * 执行 Responder
