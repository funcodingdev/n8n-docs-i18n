# HaloPSA node <a href="#halopsa-node" id="halopsa-node"></a>

使用 HaloPSA node 自动化 HaloPSA 中的工作，并将 HaloPSA 与其他应用集成。n8n 内置支持大量 HaloPSA 功能，包括创建、更新、删除和获取 client、site 与 ticket。

在本页中，你可以找到 HaloPSA node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [HaloPSA credentials](../credentials/halopsa.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Client
    * 创建 client
    * 删除 client
    * 获取 client
    * 获取所有 client
    * 更新 client
* Site
    * 创建 site
    * 删除 site
    * 获取 site
    * 获取所有 site
    * 更新 site
* Ticket
    * 创建 ticket
    * 删除 ticket
    * 获取 ticket
    * 获取所有 ticket
    * 更新 ticket
* User
    * 创建 user
    * 删除 user
    * 获取 user
    * 获取所有 user
    * 更新 user
