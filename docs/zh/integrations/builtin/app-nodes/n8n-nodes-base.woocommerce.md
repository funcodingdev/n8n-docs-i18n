# WooCommerce node <a href="#woocommerce-node" id="woocommerce-node"></a>

使用 WooCommerce node 自动化 WooCommerce 中的工作，并将 WooCommerce 与其他应用集成。n8n 内置支持大量 WooCommerce 功能，包括创建和删除客户、订单与产品。

在本页中，你可以找到 WooCommerce node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [WooCommerce credentials](../credentials/woocommerce.md)。
{% endhint %}
