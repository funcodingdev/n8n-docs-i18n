# Google Business Profile node <a href="#google-business-profile-node" id="google-business-profile-node"></a>

使用 Google Business Profile node 自动化 Google Business Profile 中的工作，并将 Google Business Profile 与其他应用集成。n8n 内置支持大量 Google Business Profile 功能，包括创建、更新和删除 post 与 review。

在本页中，你可以找到 Google Business Profile node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/google/README.md)找到此 node 的身份验证信息。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Post
    * Create
    * Delete
    * Get
    * Get Many
    * Update
* Review
    * Delete Reply
    * Get
    * Get Many
    * Reply
