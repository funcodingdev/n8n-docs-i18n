# Microsoft SQL node <a href="#microsoft-sql-node" id="microsoft-sql-node"></a>

使用 Microsoft SQL node 自动化 Microsoft SQL 中的工作，并将 Microsoft SQL 与其他应用集成。n8n 内置支持大量 Microsoft SQL 功能，包括执行 SQL 查询，以及向数据库插入 row。

在本页中，你可以找到 Microsoft SQL node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Microsoft SQL credentials](../credentials/microsoftsql.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 执行 SQL 查询
* 在数据库中插入 row
* 在数据库中更新 row
* 在数据库中删除 row
