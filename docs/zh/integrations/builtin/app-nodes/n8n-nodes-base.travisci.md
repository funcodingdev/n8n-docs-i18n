# Travis CI node <a href="#travis-ci-node" id="travis-ci-node"></a>

使用 Travis CI node 自动化 Travis CI 中的工作，并将 Travis CI 与其他应用集成。n8n 内置支持大量 Travis CI 功能，包括取消和获取 build。

在本页中，你可以找到 Travis CI node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Travis CI credentials](../credentials/travisci.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Build
    * 取消 build
    * 获取 build
    * 获取所有 build
    * 重新启动 build
    * 触发 build
