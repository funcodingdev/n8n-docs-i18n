# Microsoft Graph Security node <a href="#microsoft-graph-security-node" id="microsoft-graph-security-node"></a>

使用 Microsoft Graph Security node 自动化 Microsoft Graph Security 中的工作，并将 Microsoft Graph Security 与其他应用集成。n8n 内置支持大量 Microsoft Graph Security 功能，包括获取和更新 score 与 profile。

在本页中，你可以找到 Microsoft Graph Security node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Microsoft credentials](../credentials/microsoft.md)。
{% endhint %}

{% hint style="info" %}
**政府云支持**

如果你使用 government cloud tenant（US Government、US Government DOD 或 China），请确保在 Microsoft credentials 配置中选择合适的 **Microsoft Graph API Base URL**。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Secure Score
    * Get
    * Get All
* Secure Score Control Profile
    * Get
    * Get All
    * Update
