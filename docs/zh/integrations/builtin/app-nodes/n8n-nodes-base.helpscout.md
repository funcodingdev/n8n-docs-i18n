# Help Scout node <a href="#help-scout-node" id="help-scout-node"></a>

使用 Help Scout node 自动化 Help Scout 中的工作，并将 Help Scout 与其他应用集成。n8n 内置支持大量 Help Scout 功能，包括创建、更新、删除和获取 conversation 与 customer。

在本页中，你可以找到 Help Scout node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Help Scout credentials](../credentials/helpscout.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Conversation
    * 创建新 conversation
    * 删除 conversation
    * 获取 conversation
    * 获取所有 conversation
* Customer
    * 创建新 customer
    * 获取 customer
    * 获取所有 customer
    * 获取 customer property definition
    * 更新 customer
* Mailbox
    * 获取 mailbox 数据
    * 获取所有 mailbox
* Thread
    * 创建新 chat thread
    * 获取所有 chat thread
