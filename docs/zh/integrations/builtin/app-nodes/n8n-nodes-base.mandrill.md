# Mandrill node <a href="#mandrill-node" id="mandrill-node"></a>

使用 Mandrill node 自动化 Mandrill 中的工作，并将 Mandrill 与其他应用集成。n8n 支持使用 Mandrill 基于 template 或 HTML 发送 message。

在本页中，你可以找到 Mandrill node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Mandrill credentials](../credentials/mandrill.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Message
    * 基于 template 发送 message。
    * 基于 HTML 发送 message。
