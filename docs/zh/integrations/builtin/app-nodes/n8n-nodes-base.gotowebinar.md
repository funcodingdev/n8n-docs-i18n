# GoToWebinar node <a href="#gotowebinar-node" id="gotowebinar-node"></a>

使用 GoToWebinar node 自动化 GoToWebinar 中的工作，并将 GoToWebinar 与其他应用集成。n8n 内置支持大量 GoToWebinar 功能，包括创建、获取和删除 attendee、organizer 与 registrant。

在本页中，你可以找到 GoToWebinar node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [GoToWebinar credentials](../credentials/gotowebinar.md)。
{% endhint %}
