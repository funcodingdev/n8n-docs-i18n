# Linear node <a href="#linear-node" id="linear-node"></a>

使用 Linear node 自动化 Linear 中的工作，并将 Linear 与其他应用集成。n8n 内置支持大量 Linear 功能，包括创建、更新、删除和获取 issue。

在本页中，你可以找到 Linear node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Linear credentials](../credentials/linear.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Comment
    * 添加 Comment
* Issue
    * 添加 Link
    * Create
    * Delete
    * Get
    * Get Many
    * Update
