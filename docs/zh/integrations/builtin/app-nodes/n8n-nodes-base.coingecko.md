# CoinGecko node <a href="#coingecko-node" id="coingecko-node"></a>

使用 CoinGecko node 自动化 CoinGecko 中的工作，并将 CoinGecko 与其他应用集成。n8n 内置支持大量 CoinGecko 功能，包括获取 coin 和 event。

在本页中，你可以找到 CoinGecko node 支持的操作列表，以及更多资源链接。

{% include "https://app.gitbook.com/s/GixZThfitWP21x2gQFpD/~/reusable/6vuTxJwns2nA8U7V56ij/" %}

## 操作 <a href="#operations" id="operations"></a>

* Coin
    * 获取所选货币的 candlestick 开盘价-最高价-最低价-收盘价图表
    * 获取 coin 的当前数据
    * 获取所有 coin
    * 获取某个 coin 在给定日期的历史数据（名称、价格、市场、统计信息）
    * 获取与所选货币匹配的所有交易对的价格和市场相关数据
    * 获取历史市场数据，包括价格、市值和 24 小时成交量（自动粒度）
    * 获取你需要的任何加密货币以任何其他受支持货币计价的当前价格
    * 获取 coin ticker
* Event
    * 获取所有 event
