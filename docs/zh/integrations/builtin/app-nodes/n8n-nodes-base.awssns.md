# AWS SNS node <a href="#aws-sns-node" id="aws-sns-node"></a>

使用 AWS SNS node 自动化 AWS SNS 中的工作，并将 AWS SNS 与其他应用集成。n8n 内置支持 AWS SNS 功能，包括发布消息。

在本页中，你可以找到 AWS SNS node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [AWS SNS credentials](../credentials/aws.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 向 topic 发布消息
