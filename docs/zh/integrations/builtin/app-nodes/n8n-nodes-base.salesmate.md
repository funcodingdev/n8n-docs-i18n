# Salesmate node <a href="#salesmate-node" id="salesmate-node"></a>

使用 Salesmate node 自动化 Salesmate 中的工作，并将 Salesmate 与其他应用集成。n8n 内置支持大量 Salesmate 功能，包括创建、更新、删除和获取 activity、company 与 deal。

在本页中，你可以找到 Salesmate node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Salesmate credentials](../credentials/salesmate.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Activity
    * 创建 activity
    * 删除 activity
    * 获取 activity
    * 获取所有 company
    * 更新 activity
* Company
    * 创建 company
    * 删除 company
    * 获取 company
    * 获取所有 company
    * 更新 company
* Deal
    * 创建 deal
    * 删除 deal
    * 获取 deal
    * 获取所有 deal
    * 更新 deal
