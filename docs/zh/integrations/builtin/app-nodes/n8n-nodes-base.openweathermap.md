# OpenWeatherMap node <a href="#openweathermap-node" id="openweathermap-node"></a>

使用 OpenWeatherMap node 自动化 OpenWeatherMap 中的工作，并将 OpenWeatherMap 与其他应用集成。n8n 支持通过 OpenWeatherMap 检索当前和未来天气数据。

在本页中，你可以找到 OpenWeatherMap node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [OpenWeatherMap credentials](../credentials/openweathermap.md)。
{% endhint %}

* 返回当前天气数据
* 返回未来 5 天的天气数据
