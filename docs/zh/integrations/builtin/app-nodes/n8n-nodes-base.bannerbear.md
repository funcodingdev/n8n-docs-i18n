# Bannerbear node <a href="#bannerbear-node" id="bannerbear-node"></a>

使用 Bannerbear node 自动化 Bannerbear 中的工作，并将 Bannerbear 与其他应用集成。n8n 内置支持大量 Bannerbear 功能，包括创建和获取 image 与 template。

在本页中，你可以找到 Bannerbear node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Bannerbear credentials](../credentials/bannerbear.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Image
    * 创建 image
    * 获取 image
* Template
    * 获取 template
    * 获取所有 template
