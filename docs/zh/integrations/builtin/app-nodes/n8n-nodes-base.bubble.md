# Bubble node <a href="#bubble-node" id="bubble-node"></a>

使用 Bubble node 自动化 Bubble 中的工作，并将 Bubble 与其他应用集成。n8n 内置支持大量 Bubble 功能，包括创建、删除、获取和更新 object。

在本页中，你可以找到 Bubble node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Bubble credentials](../credentials/bubble.md)。
{% endhint %}
