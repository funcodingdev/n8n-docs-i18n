# RabbitMQ node <a href="#rabbitmq-node" id="rabbitmq-node"></a>

使用 RabbitMQ node 自动化 RabbitMQ 中的工作，并将 RabbitMQ 与其他应用集成。n8n 内置支持大量 RabbitMQ 功能，包括接收和转发消息。

在本页中，你可以找到 RabbitMQ node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [RabbitMQ credentials](../credentials/rabbitmq.md)。
{% endhint %}
