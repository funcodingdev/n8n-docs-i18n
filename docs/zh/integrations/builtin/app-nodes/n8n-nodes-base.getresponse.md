# GetResponse node <a href="#getresponse-node" id="getresponse-node"></a>

使用 GetResponse node 自动化 GetResponse 中的工作，并将 GetResponse 与其他应用集成。n8n 内置支持大量 GetResponse 功能，包括创建、更新、删除和获取 contact。

在本页中，你可以找到 GetResponse node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [GetResponse credentials](../credentials/getresponse.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Contact
    * 创建新 contact
    * 删除 contact
    * 获取 contact
    * 获取所有 contact
    * 更新 contact property
