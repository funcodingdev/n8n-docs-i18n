# Venafi TLS Protect Datacenter node <a href="#venafi-tls-protect-datacenter-node" id="venafi-tls-protect-datacenter-node"></a>

使用 Venafi TLS Protect Datacenter node 自动化 Venafi TLS Protect Datacenter 中的工作，并将 Venafi TLS Protect Datacenter 与其他应用集成。n8n 内置支持大量 Venafi TLS Protect Datacenter 功能，包括创建、删除和获取证书。

在本页中，你可以找到 Venafi TLS Protect Datacenter node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Venafi TLS Protect Datacenter credentials](../credentials/venafitlsprotectdatacenter.md)。
{% endhint %}
