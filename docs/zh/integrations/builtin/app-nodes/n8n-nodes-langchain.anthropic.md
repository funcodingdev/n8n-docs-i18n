# Anthropic node <a href="#anthropic-node" id="anthropic-node"></a>

使用 Anthropic node 自动化 Anthropic 中的工作，并将 Anthropic 与其他应用集成。n8n 内置支持大量 Anthropic 功能，包括分析、上传、获取和删除文档、文件与图片，以及生成、改进或模板化 prompt。

在本页中，你可以找到 Anthropic node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/anthropic.md)找到此 node 的身份验证信息。
{% endhint %}


## 操作 <a href="#operations" id="operations"></a>

* 文档：
	* 分析文档：接收文档并回答有关文档的问题。
* 文件：
	* 上传文件：将文件上传到 Anthropic API，供之后使用。
	* 获取文件元数据：从 Anthropic API 获取文件的元数据。
	* 列出文件：列出 Anthropic API 中的文件。
	* 删除文件：从 Anthropic API 删除文件。
* 图片：
	* 分析图片：接收图片并回答有关图片的问题。
* Prompt：
	* 生成 Prompt：为模型生成 prompt。
	* 改进 Prompt：改进模型的 prompt。
	* 模板化 Prompt：为模型模板化 prompt。
* 文本：
	* 向模型发送消息：使用 Anthropic 模型创建补全。
