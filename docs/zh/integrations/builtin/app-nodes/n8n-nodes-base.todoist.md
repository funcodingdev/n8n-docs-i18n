# Todoist node <a href="#todoist-node" id="todoist-node"></a>

使用 Todoist node 自动化 Todoist 中的工作，并将 Todoist 与其他应用集成。n8n 内置支持大量 Todoist 功能，包括创建、更新、删除和获取 task。

在本页中，你可以找到 Todoist node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Todoist credentials](../credentials/todoist.md)。
{% endhint %}

    * 更新 task
