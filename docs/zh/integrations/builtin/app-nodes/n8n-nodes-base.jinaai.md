# Jina AI node <a href="#jina-ai-node" id="jina-ai-node"></a>

使用 Jina AI node 自动化 Jina AI 中的工作，并将 Jina AI 与其他应用集成。n8n 内置支持大量 Jina AI 功能。

在本页中，你可以找到 Jina AI node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/jinaai.md)找到此 node 的身份验证信息。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* **Reader**：
    * **Read**：从 URL 获取内容，并将其转换为干净、适合 LLM 的格式。
    * **Search**：使用 Jina AI 执行网页搜索，并以干净、适合 LLM 的格式返回排名靠前的结果。
* **Research**：
    * **Deep Research**：研究一个主题并生成结构化研究报告。
