# Cockpit node <a href="#cockpit-node" id="cockpit-node"></a>

使用 Cockpit node 自动化 Cockpit 中的工作，并将 Cockpit 与其他应用集成。n8n 内置支持大量 Cockpit 功能，包括创建 collection entry、存储表单提交数据，以及获取 singleton。

在本页中，你可以找到 Cockpit node 支持的操作列表，以及更多资源链接。

## Credentials <a href="#credentials" id="credentials"></a>

有关设置身份验证的指导，请参阅 [Cockpit credentials](../credentials/cockpit.md)。

## 操作 <a href="#operations" id="operations"></a>

* Collection
    * 创建 collection entry
    * 获取所有 collection entry
    * 更新 collection entry
* Form
    * 存储表单提交数据
* Singleton
    * 获取 singleton
