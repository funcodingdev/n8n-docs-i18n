# Disqus node <a href="#disqus-node" id="disqus-node"></a>

使用 Disqus node 自动化 Disqus 中的工作，并将 Disqus 与其他应用集成。n8n 内置支持大量 Disqus 功能，包括返回 forum。

在本页中，你可以找到 Disqus node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Disqus credentials](../credentials/disqus.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Forum
    * 返回 forum 详情
    * 返回 forum 中的 category 列表
    * 返回 forum 中的 thread 列表
    * 返回 forum 中的 post 列表
