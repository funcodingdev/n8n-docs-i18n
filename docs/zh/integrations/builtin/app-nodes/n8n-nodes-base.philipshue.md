# Philips Hue node <a href="#philips-hue-node" id="philips-hue-node"></a>

使用 Philips Hue node 自动化 Philips Hue 中的工作，并将 Philips Hue 与其他应用集成。n8n 内置支持大量 Philips Hue 功能，包括删除、检索和更新 light。

在本页中，你可以找到 Philips Hue node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Philips Hue credentials](../credentials/philipshue.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Light
    * 删除 light
    * 检索 light
    * 检索所有 light
    * 更新 light
