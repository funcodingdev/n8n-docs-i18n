# Xero node <a href="#xero-node" id="xero-node"></a>

使用 Xero node 自动化 Xero 中的工作，并将 Xero 与其他应用集成。n8n 内置支持大量 Xero 功能，包括创建、更新和获取联系人与发票。

在本页中，你可以找到 Xero node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Xero credentials](../credentials/xero.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 联系人
    * 创建联系人
    * 获取联系人
    * 获取所有联系人
    * 更新联系人
* 发票
    * 创建发票
    * 获取发票
    * 获取所有发票
    * 更新发票
