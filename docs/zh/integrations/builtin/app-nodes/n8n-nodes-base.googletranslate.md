# Google Translate node <a href="#google-translate-node" id="google-translate-node"></a>

使用 Google Translate node 自动化 Google Translate 中的工作，并将 Google Translate 与其他应用集成。n8n 内置支持大量 Google Translate 功能，包括翻译语言。

在本页中，你可以找到 Google Translate node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Google Translate credentials](../credentials/google/README.md)。
{% endhint %}
