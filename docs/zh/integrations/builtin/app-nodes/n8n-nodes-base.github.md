# GitHub node <a href="#github-node" id="github-node"></a>

使用 GitHub node 自动化 GitHub 中的工作，并将 GitHub 与其他应用集成。n8n 内置支持大量 GitHub 功能，包括创建、更新、删除和编辑 file、repository、issue、release 与 user。

在本页中，你可以找到 GitHub node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [GitHub credentials](../credentials/github.md)。
{% endhint %}
