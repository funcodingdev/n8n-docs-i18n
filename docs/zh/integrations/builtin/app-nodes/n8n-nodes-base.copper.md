# Copper node <a href="#copper-node" id="copper-node"></a>

使用 Copper node 自动化 Copper 中的工作，并将 Copper 与其他应用集成。n8n 内置支持大量 Copper 功能，包括获取、更新、删除和创建 company、customer source、lead、project 与 task。

在本页中，你可以找到 Copper node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Copper credentials](../credentials/copper.md)。
{% endhint %}
