# Netscaler ADC node <a href="#netscaler-adc-node" id="netscaler-adc-node"></a>

使用 Netscaler ADC node 自动化 Netscaler ADC 中的工作，并将 Netscaler ADC 与其他应用集成。n8n 内置支持大量 Netscaler ADC 功能，包括创建和安装证书与文件。

在本页中，你可以找到 Netscaler ADC node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Netscaler ADC credentials](../credentials/netscaleradc.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Certificate
    * 创建
    * 安装
* File
    * 删除
    * 下载
    * 上传
