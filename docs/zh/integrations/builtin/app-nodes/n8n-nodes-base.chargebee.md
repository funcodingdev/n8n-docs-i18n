# Chargebee node <a href="#chargebee-node" id="chargebee-node"></a>

使用 Chargebee node 自动化 Chargebee 中的工作，并将 Chargebee 与其他应用集成。n8n 内置支持大量 Chargebee 功能，包括创建 customer、返回 invoice 和取消 subscription。

在本页中，你可以找到 Chargebee node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Chargebee credentials](../credentials/chargebee.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Customer
    * 创建 customer
* Invoice
    * 返回 invoice
    * 获取 invoice PDF 的 URL
* Subscription
    * 取消 subscription
    * 删除 subscription
