# Grafana node <a href="#grafana-node" id="grafana-node"></a>

使用 Grafana node 自动化 Grafana 中的工作，并将 Grafana 与其他应用集成。n8n 内置支持大量 Grafana 功能，包括创建、更新、删除和获取 dashboard、team 与 user。

在本页中，你可以找到 Grafana node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Grafana credentials](../credentials/grafana.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Dashboard
    * 创建 dashboard
    * 删除 dashboard
    * 获取 dashboard
    * 获取所有 dashboard
    * 更新 dashboard
* Team
    * 创建 team
    * 删除 team
    * 获取 team
    * 检索所有 team
    * 更新 team
* Team Member
    * 向 team 添加 member
    * 检索所有 team member
    * 从 team 中移除 member
* User
    * 从当前 organization 删除 user
    * 检索当前 organization 中的所有 user
    * 更新当前 organization 中的 user
