# MQTT node <a href="#mqtt-node" id="mqtt-node"></a>

使用 MQTT node 自动化 MQTT 中的工作，并将 MQTT 与其他应用集成。n8n 支持通过 MQTT 传输消息。

在本页中，你可以找到 MQTT node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [MQTT credentials](../credentials/mqtt.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

使用 MQTT node 发送消息。你可以设置消息 topic，并选择是否将 node 输入数据作为消息的一部分发送。

[浏览 MQTT node 文档集成模板](https://n8n.io/integrations/mqtt)或[搜索所有模板](https://n8n.io/workflows/)
