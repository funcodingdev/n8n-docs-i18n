# Taiga node <a href="#taiga-node" id="taiga-node"></a>

使用 Taiga node 自动化 Taiga 中的工作，并将 Taiga 与其他应用集成。n8n 内置支持大量 Taiga 功能，包括创建、更新、删除和获取 issue。

在本页中，你可以找到 Taiga node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Taiga credentials](../credentials/taiga.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

- **Issue**
    - 创建 issue
    - 删除 issue
    - 获取 issue
    - 获取所有 issue
    - 更新 issue
