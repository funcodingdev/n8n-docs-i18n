# Brandfetch node <a href="#brandfetch-node" id="brandfetch-node"></a>

使用 Brandfetch node 自动化 Brandfetch 中的工作，并将 Brandfetch 与其他应用集成。n8n 内置支持大量 Brandfetch 功能，包括返回公司的信息。

在本页中，你可以找到 Brandfetch node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Brandfetch credentials](../credentials/brandfetch.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 返回公司的颜色
* 返回公司的数据
* 返回公司的字体
* 返回公司的行业
* 返回公司的 logo 和 icon
