# AWS DynamoDB node <a href="#aws-dynamodb-node" id="aws-dynamodb-node"></a>

使用 AWS DynamoDB node 自动化 AWS DynamoDB 中的工作，并将 AWS DynamoDB 与其他应用集成。n8n 内置支持大量 AWS DynamoDB 功能，包括在数据库中创建、读取、更新和删除 item 与 record。

在本页中，你可以找到 AWS DynamoDB node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [AWS credentials](../credentials/aws.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Item
    * 创建新 record，或在当前 record 已存在时更新它（upsert/put）
    * 删除 item
    * 获取 item
    * 获取所有 item
