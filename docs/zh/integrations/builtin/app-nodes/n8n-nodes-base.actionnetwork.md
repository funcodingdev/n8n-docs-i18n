# Action Network node <a href="#action-network-node" id="action-network-node"></a>

使用 Action Network node 自动化 Action Network 中的工作，并将 Action Network 与其他应用集成。n8n 内置支持大量 Action Network 功能，包括创建、更新和删除 event、person、tag 与 signature。

在本页中，你可以找到 Action Network node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Action Network credentials](../credentials/actionnetwork.md)。
{% endhint %}
