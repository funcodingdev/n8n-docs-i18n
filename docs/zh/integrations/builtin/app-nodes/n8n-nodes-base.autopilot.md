# Autopilot node <a href="#autopilot-node" id="autopilot-node"></a>

使用 Autopilot node 自动化 Autopilot 中的工作，并将 Autopilot 与其他应用集成。n8n 内置支持大量 Autopilot 功能，包括创建、删除和更新 contact，以及将 contact 添加到 list。

在本页中，你可以找到 Autopilot node 支持的操作列表，以及更多资源链接。

{% hint style="warning" %}
**Autopilot 品牌变更**

Autopilot 已变为 Ortto。Autopilot credential 和 node 仅兼容 Autopilot，不兼容新的 Ortto API。
{% endhint %}
{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Autopilot credentials](../credentials/autopilot.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Contact
    * Create/Update a contact
    * Delete a contact
    * Get a contact
    * Get all contacts
* Contact Journey
    * Add contact to list
* Contact List
    * Add contact to list
    * Check if contact is on list
    * Get all contacts on list
    * Remove a contact from a list
* List
    * Create a list
    * Get all lists
