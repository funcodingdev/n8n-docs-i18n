# AWS Lambda node <a href="#aws-lambda-node" id="aws-lambda-node"></a>

使用 AWS Lambda node 自动化 AWS Lambda 中的工作，并将 AWS Lambda 与其他应用集成。n8n 内置支持大量 AWS Lambda 功能，包括调用 function。

在本页中，你可以找到 AWS Lambda node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [AWS Lambda credentials](../credentials/aws.md)。
{% endhint %}
