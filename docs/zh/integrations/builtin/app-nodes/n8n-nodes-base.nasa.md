# NASA node <a href="#nasa-node" id="nasa-node"></a>

使用 NASA node 自动化 NASA 中的工作，并将 NASA 与其他应用集成。n8n 内置支持大量 NASA 功能，包括检索图像和数据。

在本页中，你可以找到 NASA node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [NASA credentials](../credentials/nasa.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Astronomy Picture of the Day
    * 获取每日天文图片
* Asteroid Neo-Feed
    * 根据小行星最接近地球的日期检索小行星列表
* Asteroid Neo-Lookup
    * 根据 NASA SPK-ID 查找小行星
* Asteroid Neo-Browse
    * 浏览完整小行星数据集
* DONKI Coronal Mass Ejection
    * 检索 DONKI 日冕物质抛射数据
* DONKI Interplanetary Shock
    * 检索 DONKI 行星际激波数据
* DONKI Solar Flare
    * 检索 DONKI 太阳耀斑数据
* DONKI Solar Energetic Particle
    * 检索 DONKI 太阳高能粒子数据
* DONKI Magnetopause Crossing
    * 检索 DONKI 磁层顶穿越数据
* DONKI Radiation Belt Enhancement
    * 检索 DONKI 辐射带增强数据
* DONKI High Speed Stream
    * 检索 DONKI 高速流数据
* DONKI WSA+EnlilSimulation
    * 检索 DONKI WSA+EnlilSimulation 数据
* DONKI Notifications
    * 检索 DONKI 通知数据
* Earth Imagery
    * 检索地球图像
* Earth Assets
    * 检索地球资产
