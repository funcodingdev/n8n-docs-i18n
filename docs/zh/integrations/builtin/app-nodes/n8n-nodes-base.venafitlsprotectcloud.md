# Venafi TLS Protect Cloud node <a href="#venafi-tls-protect-cloud-node" id="venafi-tls-protect-cloud-node"></a>

使用 Venafi TLS Protect Cloud node 自动化 Venafi TLS Protect Cloud 中的工作，并将 Venafi TLS Protect Cloud 与其他应用集成。n8n 内置支持大量 Venafi TLS Protect Cloud 功能，包括删除和下载证书，以及创建证书请求。

在本页中，你可以找到 Venafi TLS Protect Cloud node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Venafi TLS Protect Cloud credentials](../credentials/venafitlsprotectcloud.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 证书
	* 删除
	* 下载
	* 获取
	* 获取多个
	* 续订
* 证书请求
	* 创建
	* 获取
	* 获取多个
