# Unleashed Software node <a href="#unleashed-software-node" id="unleashed-software-node"></a>

使用 Unleashed Software node 自动化 Unleashed Software 中的工作，并将 Unleashed Software 与其他应用集成。n8n 内置支持大量 Unleashed Software 功能，包括获取 sales order 和 stock on hand。

在本页中，你可以找到 Unleashed Software node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Unleashed Software credentials](../credentials/unleashedsoftware.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Sales Order
    * 获取所有 sales order
* Stock On Hand
    * 获取 stock on hand
    * 获取所有 stock on hand
