# Notion node <a href="#notion-node" id="notion-node"></a>

使用 Notion node 自动化 Notion 中的工作，并将 Notion 与其他应用集成。n8n 内置支持大量 Notion 功能，包括获取和搜索数据库、创建页面以及获取用户。

在本页中，你可以找到 Notion node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials 凭据**

有关身份验证设置指南，请参阅 [Notion credentials](../../credentials/notion.md)。
{% endhint %}
