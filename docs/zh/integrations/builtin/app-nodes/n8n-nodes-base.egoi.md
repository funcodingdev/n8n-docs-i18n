# E-goi node <a href="#e-goi-node" id="e-goi-node"></a>

使用 E-goi node 自动化 E-goi 中的工作，并将 E-goi 与其他应用集成。n8n 内置支持大量 E-goi 功能，包括创建、更新、删除和获取 contact。

在本页中，你可以找到 E-goi node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [E-goi credentials](../credentials/egoi.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

**Contact**

- 创建 member
- 获取 member
- 获取所有 member
- 更新 member
