# MessageBird node <a href="#messagebird-node" id="messagebird-node"></a>

使用 MessageBird node 自动化 MessageBird 中的工作，并将 MessageBird 与其他应用集成。n8n 内置支持大量 MessageBird 功能，包括发送 message 和获取 balance。

在本页中，你可以找到 MessageBird node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [MessageBird credentials](../credentials/messagebird.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* SMS
    * 发送 text message (SMS)
* Balance
    * 获取 balance
