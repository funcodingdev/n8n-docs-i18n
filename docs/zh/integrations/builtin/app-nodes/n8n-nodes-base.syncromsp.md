# SyncroMSP node <a href="#syncromsp-node" id="syncromsp-node"></a>

使用 SyncroMSP node 自动化 SyncroMSP 中的工作，并将 SyncroMSP 与其他应用集成。n8n 内置支持大量 SyncroMSP 功能，包括创建和删除新 customer、ticket 与 contact。

在本页中，你可以找到 SyncroMSP node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [SyncroMSP credentials](../credentials/syncromsp.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Contact
    * 创建新 contact
    * 删除 contact
    * 检索 contact
    * 检索所有 contact
    * 更新 contact
* Customer
    * 创建新 customer
    * 删除 customer
    * 检索 customer
    * 检索所有 customer
    * 更新 customer
* RMM
    * 创建新 RMM Alert
    * 删除 RMM Alert
    * 检索 RMM Alert
    * 检索所有 RMM Alert
    * 静音 RMM Alert
* Ticket
    * 创建新 ticket
    * 删除 ticket
    * 检索 ticket
    * 检索所有 ticket
    * 更新 ticket
