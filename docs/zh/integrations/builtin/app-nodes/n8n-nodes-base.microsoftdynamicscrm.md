# Microsoft Dynamics CRM node <a href="#microsoft-dynamics-crm-node" id="microsoft-dynamics-crm-node"></a>

使用 Microsoft Dynamics CRM node 自动化 Microsoft Dynamics CRM 中的工作，并将 Microsoft Dynamics CRM 与其他应用集成。n8n 内置支持创建、更新、删除和获取 Microsoft Dynamics CRM account。

在本页中，你可以找到 Microsoft Dynamics CRM node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Microsoft credentials](../credentials/microsoft.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Account
    * Create
    * Delete
    * Get
    * Get All
    * Update
