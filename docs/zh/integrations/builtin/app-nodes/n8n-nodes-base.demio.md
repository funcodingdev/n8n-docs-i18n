# Demio node <a href="#demio-node" id="demio-node"></a>

使用 Demio node 自动化 Demio 中的工作，并将 Demio 与其他应用集成。n8n 内置支持大量 Demio 功能，包括获取和注册 event 与 report。

在本页中，你可以找到 Demio node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Demio credentials](../credentials/demio.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Event
    * 获取 event
    * 获取所有 event
    * 将某人注册到 event
* Report
    * 获取 event report
