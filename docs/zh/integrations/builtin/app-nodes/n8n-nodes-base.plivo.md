# Plivo node <a href="#plivo-node" id="plivo-node"></a>

使用 Plivo node 自动化 Plivo 中的工作，并将 Plivo 与其他应用集成。n8n 内置支持大量 Plivo 功能，包括拨打电话和发送 SMS/MMS。

在本页中，你可以找到 Plivo node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Plivo credentials](../credentials/plivo.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Call
    * 发起语音呼叫
* MMS
    * 发送 MMS 消息（仅限美国/加拿大）
* SMS
    * 发送 SMS 消息。
