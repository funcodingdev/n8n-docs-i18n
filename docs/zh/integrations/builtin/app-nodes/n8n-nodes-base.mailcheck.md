# Mailcheck node <a href="#mailcheck-node" id="mailcheck-node"></a>

使用 Mailcheck node 自动化 Mailcheck 中的工作，并将 Mailcheck 与其他应用集成。n8n 内置支持大量 Mailcheck 功能，包括检查 email。

在本页中，你可以找到 Mailcheck node 支持的操作列表，以及更多资源链接。
