# Discourse node <a href="#discourse-node" id="discourse-node"></a>

使用 Discourse node 自动化 Discourse 中的工作，并将 Discourse 与其他应用集成。n8n 内置支持大量 Discourse 功能，包括创建、获取、更新和移除 category、group、post 与 user。

在本页中，你可以找到 Discourse node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Discourse credentials](../credentials/discourse.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Category
    * 创建 category
    * 获取所有 category
    * 更新 category
* Group
    * 创建 group
    * 获取 group
    * 获取所有 group
    * 更新 group
* Post
    * 创建 post
    * 获取 post
    * 获取所有 post
    * 更新 post
* User
    * 创建 user
    * 获取 user
    * 获取所有 user
* User Group
    * 将 user 添加到 group
    * 从 group 中移除 user
