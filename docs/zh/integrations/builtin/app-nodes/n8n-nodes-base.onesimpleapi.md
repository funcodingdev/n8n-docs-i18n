# One Simple API node <a href="#one-simple-api-node" id="one-simple-api-node"></a>

使用 One Simple API node 自动化 One Simple API 中的工作，并将 One Simple API 与其他应用集成。n8n 内置支持大量 One Simple API 功能，包括获取 profile、检索信息和生成 utility。

在本页中，你可以找到 One Simple API node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [One Simple API credentials](../credentials/onesimpleapi.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Information
    * 在不同货币之间转换数值
    * 从 URL 检索图像 metadata
* Social Profile
    * 获取 Instagram profile 的详细信息
    * 获取 Spotify Artist 的详细信息
* Utility
    * 展开缩短后的 URL
    * 生成 QR Code
    * 验证 email address
* Website
    * 从网页生成 PDF
    * 从网站获取 SEO 信息
    * 从网页创建截图
