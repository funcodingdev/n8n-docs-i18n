# Twilio node <a href="#twilio-node" id="twilio-node"></a>

使用 Twilio node 自动化 Twilio 中的工作，并将 Twilio 与其他应用集成。n8n 支持通过 Twilio 发送 MMS/SMS 和 WhatsApp message。

在本页中，你可以找到 Twilio node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Twilio credentials](../credentials/twilio.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* SMS
    * 发送 SMS/MMS/WhatsApp message
* Call
    * 使用 text-to-speech 拨打电话并朗读 message
