# AMQP Sender node <a href="#amqp-sender-node" id="amqp-sender-node"></a>

使用 AMQP Sender node 自动化 AMQP Sender 中的工作，并将 AMQP Sender 与其他应用集成。n8n 内置支持 AMQP Sender 的功能，包括发送消息。

在本页中，你可以找到 AMQP Sender node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [AMQP Sender credentials](../credentials/amqp.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

- Send message
