# X (Formerly Twitter) node <a href="#x-formerly-twitter-node" id="x-formerly-twitter-node"></a>

使用 X node 自动化 X 中的工作，并将 X 与其他应用集成。n8n 内置支持大量 X 功能，包括创建 direct message，以及删除、搜索、点赞和转发 tweet。

在本页中，你可以找到 X node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [X credentials](../credentials/twitter.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Direct Message
    * 创建 direct message
* Tweet
    * 创建或回复 tweet
    * 删除 tweet
    * 搜索 tweet
    * 点赞 tweet
    * 转发 tweet
* User
    * 获取 user
* List
    * 向 list 添加 member
