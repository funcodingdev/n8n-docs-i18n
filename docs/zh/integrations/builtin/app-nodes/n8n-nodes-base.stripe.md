# Stripe node <a href="#stripe-node" id="stripe-node"></a>

使用 Stripe node 自动化 Stripe 中的工作，并将 Stripe 与其他应用集成。n8n 内置支持大量 Stripe 功能，包括获取 balance、创建 charge 和 meter event，以及删除 customer。

在本页中，你可以找到 Stripe node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Stripe credentials](../credentials/stripe.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Balance
    * 获取 balance
* Charge
    * 创建 charge
    * 获取 charge
    * 获取所有 charge
    * 更新 charge
* Coupon
    * 创建 coupon
    * 获取所有 coupon
* Customer
    * 创建 customer
    * 删除 customer
    * 获取 customer
    * 获取所有 customer
    * 更新 customer
* Customer Card
    * 添加 customer card
    * 获取 customer card
    * 移除 customer card
* Meter Event
    * 创建 meter event
* Source
    * 创建 source
    * 删除 source
    * 获取 source
* Token
    * 创建 token
