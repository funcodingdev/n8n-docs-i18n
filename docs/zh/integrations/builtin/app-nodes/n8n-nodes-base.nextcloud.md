# Nextcloud node <a href="#nextcloud-node" id="nextcloud-node"></a>

使用 Nextcloud node 自动化 Nextcloud 中的工作，并将 Nextcloud 与其他应用集成。n8n 内置支持大量 Nextcloud 功能，包括创建、更新、删除和获取文件与文件夹，以及检索和邀请用户。

在本页中，你可以找到 Nextcloud node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Nextcloud credentials](../credentials/nextcloud.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* File
    * 复制文件
    * 删除文件
    * 下载文件
    * 移动文件
    * 共享文件
    * 上传文件
* Folder
    * 复制文件夹
    * 创建文件夹
    * 删除文件夹
    * 返回给定文件夹的内容
    * 移动文件夹
    * 共享文件夹
* User
    * 邀请用户加入 Nextcloud organization
    * 删除用户。
    * 检索单个用户的信息。
    * 检索用户列表。
    * 编辑与用户相关的属性。
