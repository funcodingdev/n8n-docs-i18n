# Google Cloud Storage node <a href="#google-cloud-storage-node" id="google-cloud-storage-node"></a>

使用 Google Cloud Storage node 自动化 Google Cloud Storage 中的工作，并将 Google Cloud Storage 与其他应用集成。n8n 内置支持大量 Google Cloud Storage 功能，包括创建、更新、删除和获取 bucket 与 object。

在本页中，你可以找到 Google Cloud Storage node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Google Cloud Storage credentials](../credentials/google/README.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Bucket
    * Create
    * Delete
    * Get
    * Get Many
    * Update
* Object
    * Create
    * Delete
    * Get
    * Get Many
    * Update
