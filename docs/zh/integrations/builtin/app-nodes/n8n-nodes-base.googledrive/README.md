# Google Drive node <a href="#google-drive-node" id="google-drive-node"></a>

使用 Google Drive node 自动化 Google Drive 中的工作，并将 Google Drive 与其他应用集成。n8n 内置支持大量 Google Drive 功能，包括创建、更新、列出、删除和获取 drive、文件和文件夹。

在本页中，你可以找到 Google Drive node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials 凭据**

有关身份验证设置指南，请参阅 [Google Drive credentials](../../credentials/google/README.md)。
{% endhint %}
