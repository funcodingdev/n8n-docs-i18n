# ServiceNow node <a href="#servicenow-node" id="servicenow-node"></a>

使用 ServiceNow node 自动化 ServiceNow 中的工作，并将 ServiceNow 与其他应用集成。n8n 内置支持大量 ServiceNow 功能，包括获取 business service、department、configuration item 和 dictionary，以及创建、更新和删除 incident、user 与 table record。

在本页中，你可以找到 ServiceNow node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [ServiceNow credentials](../credentials/servicenow.md)。
{% endhint %}
