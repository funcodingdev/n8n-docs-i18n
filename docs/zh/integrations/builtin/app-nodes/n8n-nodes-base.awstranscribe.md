# AWS Transcribe node <a href="#aws-transcribe-node" id="aws-transcribe-node"></a>

使用 AWS Transcribe node 自动化 AWS Transcribe 中的工作，并将 AWS Transcribe 与其他应用集成。n8n 内置支持大量 AWS Transcribe 功能，包括创建、删除和获取 transcription job。

在本页中，你可以找到 AWS Transcribe node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [AWS Transcribe credentials](../credentials/aws.md)。
{% endhint %}
