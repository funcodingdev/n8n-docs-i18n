# Oura node <a href="#oura-node" id="oura-node"></a>

使用 Oura node 自动化 Oura 中的工作，并将 Oura 与其他应用集成。n8n 内置支持大量 Oura 功能，包括获取 profile 和 summary。

在本页中，你可以找到 Oura node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Oura credentials](../credentials/oura.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Profile
    * 获取用户个人信息。
* Summary
    * 获取用户活动 summary。
    * 获取用户 readiness summary。
    * 获取用户 sleep summary
