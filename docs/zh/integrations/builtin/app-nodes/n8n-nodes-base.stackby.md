# Stackby node <a href="#stackby-node" id="stackby-node"></a>

使用 Stackby node 自动化 Stackby 中的工作，并将 Stackby 与其他应用集成。n8n 内置支持大量 Stackby 功能，包括追加、删除、列出和读取。

在本页中，你可以找到 Stackby node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Stackby credentials](../credentials/stackby.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

- 追加
- 删除
- 列出
- 读取
