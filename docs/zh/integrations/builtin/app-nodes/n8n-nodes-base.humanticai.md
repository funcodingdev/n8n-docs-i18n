# Humantic AI node <a href="#humantic-ai-node" id="humantic-ai-node"></a>

使用 Humantic AI node 自动化 Humantic AI 中的工作，并将 Humantic AI 与其他应用集成。n8n 内置支持大量 Humantic AI 功能，包括创建、检索和更新 profile。

在本页中，你可以找到 Humantic AI node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Humantic AI credentials](../credentials/humanticai.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Profile
    * 创建 profile
    * 检索 profile
    * 更新 profile
