# Elasticsearch node <a href="#elasticsearch-node" id="elasticsearch-node"></a>

使用 Elasticsearch node 自动化 Elasticsearch 中的工作，并将 Elasticsearch 与其他应用集成。n8n 内置支持大量 Elasticsearch 功能，包括创建、更新、删除和获取 document 与 index。

在本页中，你可以找到 Elasticsearch node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Elasticsearch credentials](../credentials/elasticsearch.md)。
{% endhint %}
