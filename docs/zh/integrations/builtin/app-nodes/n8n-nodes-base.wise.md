# Wise node <a href="#wise-node" id="wise-node"></a>

使用 Wise node 自动化 Wise 中的工作，并将 Wise 与其他应用集成。n8n 内置支持大量 Wise 功能，包括获取 profile、汇率和收款人。

在本页中，你可以找到 Wise node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Wise credentials](../credentials/wise.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 账户
    * 获取此用户所有账户货币的余额
    * 获取此用户 borderless account 中的货币
    * 获取此用户 borderless account 的对账单
* 汇率
    * 获取
* 资料
    * 获取
    * 获取全部
* 收款人
    * 获取全部
* 报价
    * 创建
    * 获取
* 转账
    * 创建
    * 删除
    * 执行
    * 获取
    * 获取全部
