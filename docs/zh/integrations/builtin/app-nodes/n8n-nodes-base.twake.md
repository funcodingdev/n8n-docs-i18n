# Twake node <a href="#twake-node" id="twake-node"></a>

使用 Twake node 自动化 Twake 中的工作，并将 Twake 与其他应用集成。n8n 支持通过 Twake 发送 message。

在本页中，你可以找到 Twake node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Twake credentials](../credentials/twake.md)。
{% endhint %}

    * 发送 message
