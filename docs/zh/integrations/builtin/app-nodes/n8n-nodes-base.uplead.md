# UpLead node <a href="#uplead-node" id="uplead-node"></a>

使用 UpLead node 自动化 UpLead 中的工作，并将 UpLead 与其他应用集成。n8n 支持多个 UpLead 操作，包括获取 company information。

在本页中，你可以找到 UpLead node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [UpLead credentials](../credentials/uplead.md)。
{% endhint %}
