# Pushcut node <a href="#pushcut-node" id="pushcut-node"></a>

使用 Pushcut node 自动化 Pushcut 中的工作，并将 Pushcut 与其他应用集成。n8n 支持通过 Pushcut 发送通知。

在本页中，你可以找到 Pushcut node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Pushcut credentials](../credentials/pushcut.md)。
{% endhint %}

* Notification
    * 发送通知
