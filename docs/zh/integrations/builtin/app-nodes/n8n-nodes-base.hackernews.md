# Hacker News node <a href="#hacker-news-node" id="hacker-news-node"></a>

使用 Hacker News node 自动化 Hacker News 中的工作，并将 Hacker News 与其他应用集成。n8n 内置支持大量 Hacker News 功能，包括获取 article 和 user。

在本页中，你可以找到 Hacker News node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

此 node 不需要身份验证。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* All
    * 获取所有 item
* Article
    * 获取 Hacker News article
* User
    * 获取 Hacker News user
