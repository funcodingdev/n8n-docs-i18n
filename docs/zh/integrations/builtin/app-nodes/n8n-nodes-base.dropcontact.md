# Dropcontact node <a href="#dropcontact-node" id="dropcontact-node"></a>

使用 Dropcontact node 自动化 Dropcontact 中的工作，并将 Dropcontact 与其他应用集成。n8n 内置支持大量 Dropcontact 功能，包括获取 contact。

在本页中，你可以找到 Dropcontact node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Dropcontact credentials](../credentials/dropcontact.md)。
{% endhint %}

- Fetch Request
