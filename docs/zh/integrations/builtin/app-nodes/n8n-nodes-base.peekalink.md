# Peekalink node <a href="#peekalink-node" id="peekalink-node"></a>

使用 Peekalink node 自动化 Peekalink 中的工作，并将 Peekalink 与其他应用集成。n8n 支持通过 Peekalink 检查和查看链接。

在本页中，你可以找到 Peekalink node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Peekalink credentials](../credentials/peekalink.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 检查给定链接是否有可用 preview
* 返回链接的 preview
