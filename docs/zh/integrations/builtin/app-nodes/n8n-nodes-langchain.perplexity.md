# Perplexity node <a href="#perplexity-node" id="perplexity-node"></a>

使用 Perplexity node 自动化 Perplexity 中的工作，并将 Perplexity 与其他应用集成。n8n 内置支持向模型发送消息。

在本页中，你可以找到 Perplexity node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/perplexity.md)找到此 node 的身份验证信息。
{% endhint %}


## 操作 <a href="#operations" id="operations"></a>

* **向模型发送消息**：为给定文本创建一个或多个补全。
