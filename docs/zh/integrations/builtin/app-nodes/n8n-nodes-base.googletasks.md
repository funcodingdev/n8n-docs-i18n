# Google Tasks node <a href="#google-tasks-node" id="google-tasks-node"></a>

使用 Google Tasks node 自动化 Google Tasks 中的工作，并将 Google Tasks 与其他应用集成。n8n 内置支持大量 Google Tasks 功能，包括添加、更新和检索 contact。

在本页中，你可以找到 Google Tasks node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Google Tasks credentials](../credentials/google/README.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Task
    * 将 task 添加到 task list
    * 删除 task
    * 检索 task
    * 从 task list 检索所有 task
    * 更新 task
