# Discord node <a href="#discord-node" id="discord-node"></a>

使用 Discord node 自动化 Discord 中的工作，并将 Discord 与其他 applications 集成。n8n 内置支持大量 Discord 功能，包括在 Discord channel 中发送 messages 和管理 channels。

在本页中，你会看到 Discord node 支持的 operations 列表，以及更多资源链接。

{% hint style="info" %}
**Credentials 凭据**

有关设置身份验证的指导，请参考 [Discord credentials](../../credentials/discord.md)。
{% endhint %}
