# Customer.io node <a href="#customerio-node" id="customerio-node"></a>

使用 Customer.io node 自动化 Customer.io 中的工作，并将 Customer.io 与其他应用集成。n8n 内置支持大量 Customer.io 功能，包括创建和更新 customer、跟踪 event，以及获取 campaign。

在本页中，你可以找到 Customer.io node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Customer.io credentials](../credentials/customerio.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Customer
    * 创建/更新 customer。
    * 删除 customer。
* Event
    * 跟踪 customer event。
    * 跟踪匿名 event。
* Campaign
    * Get
    * Get All
    * Get Metrics
* Segment
    * 添加 Customer
    * 移除 Customer
