# urlscan.io node <a href="#urlscanio-node" id="urlscanio-node"></a>

使用 urlscan.io node 自动化 urlscan.io 中的工作，并将 urlscan.io 与其他应用集成。n8n 内置支持大量 urlscan.io 功能，包括获取和执行 scan。

在本页中，你可以找到 urlscan.io node 支持的操作列表，以及更多资源链接。
