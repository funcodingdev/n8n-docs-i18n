# Clearbit node <a href="#clearbit-node" id="clearbit-node"></a>

使用 Clearbit node 自动化 Clearbit 中的工作，并将 Clearbit 与其他应用集成。n8n 内置支持大量 Clearbit 功能，包括自动补全和查找 company 与 person。

在本页中，你可以找到 Clearbit node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Clearbit credentials](../credentials/clearbit.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Company
    * 自动补全 company 名称，并检索 logo 和 domain
    * 基于 email 或 domain 查找 person 和 company 数据
* Person
    * 基于 email 或 domain 查找 person 和 company 数据
