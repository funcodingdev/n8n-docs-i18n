# Storyblok node <a href="#storyblok-node" id="storyblok-node"></a>

使用 Storyblok node 自动化 Storyblok 中的工作，并将 Storyblok 与其他应用集成。n8n 内置支持大量 Storyblok 功能，包括获取、删除和发布 story。

在本页中，你可以找到 Storyblok node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Storyblok credentials](../credentials/storyblok.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

### Content API <a href="#content-api" id="content-api"></a>

- **Story**
    - 获取 story
    - 获取所有 story

### Management API <a href="#management-api" id="management-api"></a>

- **Story**
    - 删除 story
    - 获取 story
    - 获取所有 story
    - 发布 story
    - 取消发布 story
