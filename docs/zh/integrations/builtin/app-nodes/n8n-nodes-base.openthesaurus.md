# OpenThesaurus node <a href="#openthesaurus-node" id="openthesaurus-node"></a>

使用 OpenThesaurus node 自动化 OpenThesaurus 中的工作，并将 OpenThesaurus 与其他应用集成。n8n 支持查找德语单词的同义词。

在本页中，你可以找到 OpenThesaurus node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

OpenThesaurus node 不需要身份验证。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 获取德语单词的德语同义词
