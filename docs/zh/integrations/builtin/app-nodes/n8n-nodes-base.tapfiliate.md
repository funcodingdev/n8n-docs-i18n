# Tapfiliate node <a href="#tapfiliate-node" id="tapfiliate-node"></a>

使用 Tapfiliate node 自动化 Tapfiliate 中的工作，并将 Tapfiliate 与其他应用集成。n8n 内置支持大量 Tapfiliate 功能，包括创建和删除 affiliate，以及添加 affiliate metadata。

在本页中，你可以找到 Tapfiliate node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Tapfiliate credentials](../credentials/tapfiliate.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Affiliate
    * 创建 affiliate
    * 删除 affiliate
    * 按 ID 获取 affiliate
    * 获取所有 affiliate
* Affiliate Metadata
    * 向 affiliate 添加 metadata
    * 从 affiliate 移除 metadata
    * 更新 affiliate 的 metadata
* Program Affiliate
    * 将 affiliate 添加到 program
    * 批准 affiliate 加入 program
    * 拒绝 affiliate
    * 获取 program 中的 affiliate
    * 获取 program 中的所有 affiliate
