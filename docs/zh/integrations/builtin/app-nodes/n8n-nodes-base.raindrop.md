# Raindrop node <a href="#raindrop-node" id="raindrop-node"></a>

使用 Raindrop node 自动化 Raindrop 中的工作，并将 Raindrop 与其他应用集成。n8n 内置支持大量 Raindrop 功能，包括获取 user、删除 tag，以及创建、更新、删除和获取 collection 与 bookmark。

在本页中，你可以找到 Raindrop node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Raindrop credentials](../credentials/raindrop.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Bookmark
    * 创建
    * 删除
    * 获取
    * 获取全部
    * 更新
* Collection
    * 创建
    * 删除
    * 获取
    * 获取全部
    * 更新
* Tag
    * 删除
    * 获取全部
* User
    * 获取
