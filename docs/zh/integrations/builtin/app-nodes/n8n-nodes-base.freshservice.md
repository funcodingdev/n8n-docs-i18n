# Freshservice node <a href="#freshservice-node" id="freshservice-node"></a>

使用 Freshservice node 自动化 Freshservice 中的工作，并将 Freshservice 与其他应用集成。n8n 内置支持大量 Freshservice 功能，包括创建、更新、删除和获取 agent 信息与 department。

在本页中，你可以找到 Freshservice node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Freshservice credentials](../credentials/freshservice.md)。
{% endhint %}
