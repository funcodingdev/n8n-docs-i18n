# Splunk node <a href="#splunk-node" id="splunk-node"></a>

使用 Splunk node 自动化 Splunk 中的工作，并将 Splunk 与其他应用集成。n8n 内置支持大量 Splunk 功能，包括获取 fired alert report，以及删除和获取 search configuration。

在本页中，你可以找到 Splunk node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Splunk credentials](../credentials/splunk.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Fired Alert
    * 获取 fired alert report
* Search Configuration
    * 删除 search configuration
    * 获取 search configuration
    * 获取多个 search configuration
* Search Job
    * 创建 search job
    * 删除 search job
    * 获取 search job
    * 获取多个 search job
* Search Result
    * 获取多个 search result
* User
    * 创建 user
    * 删除 user
    * 获取 user
    * 获取多个 user
    * 更新 user
