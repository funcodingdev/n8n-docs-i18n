# Zoom node <a href="#zoom-node" id="zoom-node"></a>

使用 Zoom node 自动化 Zoom 中的工作，并将 Zoom 与其他应用集成。n8n 内置支持大量 Zoom 功能，包括创建、检索、删除和更新会议。

在本页中，你可以找到 Zoom node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Zoom credentials](../credentials/zoom.md)。
{% endhint %}
