# Iterable node <a href="#iterable-node" id="iterable-node"></a>

使用 Iterable node 自动化 Iterable 中的工作，并将 Iterable 与其他应用集成。n8n 内置支持大量 Iterable 功能，包括创建 user、记录 user 执行的操作，以及将 user 添加到 list 或从 list 中移除。

在本页中，你可以找到 Iterable node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Iterable credentials](../credentials/iterable.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Event
    * 记录 user 执行的操作
* User
    * 创建/更新 user
    * 删除 user
    * 获取 user
* User List
    * 将 user 添加到 list
    * 从 list 中移除 user
