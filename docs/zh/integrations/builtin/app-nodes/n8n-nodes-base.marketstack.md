# marketstack node <a href="#marketstack-node" id="marketstack-node"></a>

使用 marketstack node 自动化 marketstack 中的工作，并将 marketstack 与其他应用集成。n8n 内置支持大量 marketstack 功能，包括获取 exchange、end-of-day data 和 ticker。

在本页中，你可以找到 marketstack node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [marketstack credentials](../credentials/marketstack.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* End-of-Day Data
    * Get All
* Exchange
    * Get
* Ticker
    * Get
