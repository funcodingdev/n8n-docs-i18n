# ConvertKit node <a href="#convertkit-node" id="convertkit-node"></a>

使用 ConvertKit node 自动化 ConvertKit 中的工作，并将 ConvertKit 与其他应用集成。n8n 内置支持大量 ConvertKit 功能，包括创建和删除 custom field、获取 tag，以及添加 subscriber。

在本页中，你可以找到 ConvertKit node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [ConvertKit credentials](../credentials/convertkit.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Custom Field
    * 创建 field
    * 删除 field
    * 获取所有 field
    * 更新 field
* Form
    * 添加 subscriber
    * 获取所有 form
    * 列出 form 的 subscription，包括 subscriber 数据
* Sequence
    * 添加 subscriber
    * 获取所有 sequence
    * 获取 sequence 的所有 subscription，包括 subscriber 数据
* Tag
    * 创建 tag
    * 获取所有 tag
* Tag Subscriber
    * 为 subscriber 添加 tag
    * 列出 tag 的 subscription，包括 subscriber 数据
    * 从 subscriber 删除 tag
