# Paddle node <a href="#paddle-node" id="paddle-node"></a>

使用 Paddle node 自动化 Paddle 中的工作，并将 Paddle 与其他应用集成。n8n 内置支持大量 Paddle 功能，包括创建、更新和获取 coupon，以及获取 plan、product 和 user。

在本页中，你可以找到 Paddle node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Paddle credentials](../credentials/paddle.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Coupon
    * 创建 coupon。
    * 获取所有 coupon。
    * 更新 coupon。
* Payment
    * 获取所有 payment。
    * 重新安排 payment。
* Plan
    * 获取 plan。
    * 获取所有 plan。
* Product
    * 获取所有 product。
* User
    * 获取所有 user
