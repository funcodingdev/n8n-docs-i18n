# Odoo node <a href="#odoo-node" id="odoo-node"></a>

使用 Odoo node 自动化 Odoo 中的工作，并将 Odoo 与其他应用集成。n8n 内置支持大量 Odoo 功能，包括创建、更新、删除和获取 contract、resource 与 opportunity。

在本页中，你可以找到 Odoo node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Odoo credentials](../credentials/odoo.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Contact
    * 创建新 contact
    * 删除 contact
    * 获取 contact
    * 获取所有 contact
    * 更新 contact
* Custom Resource
    * 创建新 item
    * 删除 item
    * 获取 item
    * 获取所有 item
    * 更新 item
* Note
    * 创建新 note
    * 删除 note
    * 获取 note
    * 获取所有 note
    * 更新 note
* Opportunity
    * 创建新 opportunity
    * 删除 opportunity
    * 获取 opportunity
    * 获取所有 opportunity
    * 更新 opportunity
