# Pushover node <a href="#pushover-node" id="pushover-node"></a>

使用 Pushover node 自动化 Pushover 中的工作，并将 Pushover 与其他应用集成。n8n 支持通过 Pushover 发送推送通知。

在本页中，你可以找到 Pushover node 支持的操作列表，以及更多资源链接。
