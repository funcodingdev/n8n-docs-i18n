# Sentry.io node <a href="#sentryio-node" id="sentryio-node"></a>

使用 Sentry.io node 自动化 Sentry.io 中的工作，并将 Sentry.io 与其他应用集成。n8n 内置支持大量 Sentry.io 功能，包括创建、更新、删除和获取 issue、project 与 release，以及获取所有 event。

在本页中，你可以找到 Sentry.io node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Sentry.io credentials](../credentials/sentryio.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Event
    * 按 ID 获取 event
    * 获取所有 event
* Issue
    * 删除 issue
    * 按 ID 获取 issue
    * 获取所有 issue
    * 更新 issue
* Project
    * 创建新 project
    * 删除 project
    * 按 ID 获取 project
    * 获取所有 project
    * 更新 project
* Release
    * 创建 release
    * 删除 release
    * 按 version identifier 获取 release
    * 获取所有 release
    * 更新 release
* Organization
    * 创建 organization
    * 按 slug 获取 organization
    * 获取所有 organization
    * 更新 organization
* Team
    * 创建新 team
    * 删除 team
    * 按 slug 获取 team
    * 获取所有 team
    * 更新 team
