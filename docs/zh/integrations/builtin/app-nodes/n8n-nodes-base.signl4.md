# SIGNL4 node <a href="#signl4-node" id="signl4-node"></a>

使用 SIGNL4 node 自动化 SIGNL4 中的工作，并将 SIGNL4 与其他应用集成。n8n 支持通过 SIGNL4 发送和解决 alert。

在本页中，你可以找到 SIGNL4 node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [SIGNL4 credentials](../credentials/signl4.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Alert
    * 发送 alert
    * 解决 alert
