# AWS Comprehend node <a href="#aws-comprehend-node" id="aws-comprehend-node"></a>

使用 AWS Comprehend node 自动化 AWS Comprehend 中的工作，并将 AWS Comprehend 与其他应用集成。n8n 内置支持大量 AWS Comprehend 功能，包括识别和分析文本。

在本页中，你可以找到 AWS Comprehend node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [AWS Comprehend credentials](../credentials/aws.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

**Text**

- 识别主要语言
- 分析文本情感
