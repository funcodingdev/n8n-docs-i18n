# MISP node <a href="#misp-node" id="misp-node"></a>

使用 MISP node 自动化 MISP 中的工作，并将 MISP 与其他应用集成。n8n 内置支持大量 MISP 功能，包括创建、更新、删除和获取 event、feed 与 organisation。

在本页中，你可以找到 MISP node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [MISP credentials](../credentials/misp.md)。
{% endhint %}
