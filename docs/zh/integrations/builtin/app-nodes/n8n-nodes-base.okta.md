# Okta node <a href="#okta-node" id="okta-node"></a>

使用 Okta node 自动化 Okta 中的工作，并将 Okta 与其他应用集成。n8n 内置支持大量 Okta 功能，包括创建、更新和删除用户。

在本页中，你可以找到 Okta node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/okta.md)找到此 node 的身份验证信息。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

- User
    - 创建新用户
    - 删除现有用户
    - 获取用户详细信息
    - 获取多个用户
    - 更新现有用户
