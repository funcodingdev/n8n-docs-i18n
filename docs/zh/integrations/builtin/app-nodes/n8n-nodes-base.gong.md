# Gong node <a href="#gong-node" id="gong-node"></a>

使用 Gong node 自动化 Gong 中的工作，并将 Gong 与其他应用集成。n8n 内置支持大量 Gong 功能，包括获取一个或多个 call 与 user。

在本页中，你可以找到 Gong node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/gong.md)找到此 node 的身份验证信息。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Call
    * Get
    * Get Many
* User
    * Get
    * Get Many
