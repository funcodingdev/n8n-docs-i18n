# Google Cloud Realtime Database node <a href="#google-cloud-realtime-database-node" id="google-cloud-realtime-database-node"></a>

使用 Google Cloud Realtime Database node 自动化 Google Cloud Realtime Database 中的工作，并将 Google Cloud Realtime Database 与其他应用集成。n8n 内置支持大量 Google Cloud Realtime Database 功能，包括写入、删除、获取和追加数据库数据。

在本页中，你可以找到 Google Cloud Realtime Database node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Google Cloud Realtime Database credentials](../credentials/google/README.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 向 database 写入数据
* 从 database 删除数据
* 从 database 获取 record
* 追加到数据列表
* 更新 database 上的 item
