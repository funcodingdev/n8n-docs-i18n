# Clockify node <a href="#clockify-node" id="clockify-node"></a>

使用 Clockify node 自动化 Clockify 中的工作，并将 Clockify 与其他应用集成。n8n 内置支持大量 Clockify 功能，包括创建、更新、获取和删除 task、time entry、project 与 tag。

在本页中，你可以找到 Clockify node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Clockify credentials](../credentials/clockify.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Project
    * 创建 project
    * 删除 project
    * 获取 project
    * 获取所有 project
    * 更新 project
* Tag
    * 创建 tag
    * 删除 tag
    * 获取所有 tag
    * 更新 tag
* Task
    * 创建 task
    * 删除 task
    * 获取 task
    * 获取所有 task
    * 更新 task
* Time Entry
    * 创建 time entry
    * 删除 time entry
    * 获取 time entry
    * 更新 time entry
