# Netlify node <a href="#netlify-node" id="netlify-node"></a>

使用 Netlify node 自动化 Netlify 中的工作，并将 Netlify 与其他应用集成。n8n 内置支持大量 Netlify 功能，包括获取和取消 deployment，以及删除和获取 site。

在本页中，你可以找到 Netlify node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Netlify credentials](../credentials/netlify.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Deploy
    * 取消 deployment
    * 创建新 deployment
    * 获取 deployment
    * 获取所有 deployment
* Site
    * 删除 site
    * 获取 site
    * 返回所有 site
