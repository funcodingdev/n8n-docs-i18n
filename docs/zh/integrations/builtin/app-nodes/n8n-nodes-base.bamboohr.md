# BambooHR node <a href="#bamboohr-node" id="bamboohr-node"></a>

使用 BambooHR node 自动化 BambooHR 中的工作，并将 BambooHR 与其他应用集成。n8n 内置支持大量 BambooHR 功能，包括创建、删除、下载和获取 company report、employee document 与 file。

在本页中，你可以找到 BambooHR node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [BambooHR credentials](../credentials/bamboohr.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Company Report
    * 获取 company report
* Employee
    * 创建 employee
    * 获取 employee
    * 获取所有 employee
    * 更新 employee
* Employee Document
    * 删除 employee document
    * 下载 employee document
    * 获取所有 employee document
    * 更新 employee document
    * 上传 employee document
* File
    * 删除 company file
    * 下载 company file
    * 获取所有 company file
    * 更新 company file
    * 上传 company file
