# Box node <a href="#box-node" id="box-node"></a>

使用 Box node 自动化 Box 中的工作，并将 Box 与其他应用集成。n8n 内置支持大量 Box 功能，包括创建、复制、删除、搜索、上传和下载 file 与 folder。

在本页中，你可以找到 Box node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Box credentials](../credentials/box.md)。
{% endhint %}


## 操作 <a href="#operations" id="operations"></a>

* File
    * 复制 file
    * 删除 file
    * 下载 file
    * 获取 file
    * 搜索 file
    * 分享 file
    * 上传 file
* Folder
    * 创建 folder
    * 获取 folder
    * 删除 folder
    * 搜索 file
    * 分享 folder
    * 更新 folder
