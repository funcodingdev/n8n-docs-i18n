# Quick Base node <a href="#quick-base-node" id="quick-base-node"></a>

使用 Quick Base node 自动化 Quick Base 中的工作，并将 Quick Base 与其他应用集成。n8n 内置支持大量 Quick Base 功能，包括创建、更新、删除和获取 record，以及获取 field 和下载 file。

在本页中，你可以找到 Quick Base node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Quick Base credentials](../credentials/quickbase.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Field
    * 获取所有 field
* File
    * 删除 file
    * 下载 file
* Record
    * 创建 record
    * 删除 record
    * 获取所有 record
    * 更新 record
    * Upsert record
* Report
    * 获取 report
    * 运行 report
