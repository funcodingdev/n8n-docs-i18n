# Zoho CRM node <a href="#zoho-crm-node" id="zoho-crm-node"></a>

使用 Zoho CRM node 自动化 Zoho CRM 中的工作，并将 Zoho CRM 与其他应用集成。n8n 内置支持大量 Zoho CRM 功能，包括创建和删除账户、联系人与交易。

在本页中，你可以找到 Zoho CRM node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Zoho CRM credentials](../credentials/zoho.md)。
{% endhint %}
