# AWS Textract node <a href="#aws-textract-node" id="aws-textract-node"></a>

使用 AWS Textract node 自动化 AWS Textract 中的工作，并将 AWS Textract 与其他应用集成。n8n 内置支持大量 AWS Textract 功能，包括分析 invoice。

在本页中，你可以找到 AWS Textract node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [AWS Textract credentials](../credentials/aws.md)。
{% endhint %}
