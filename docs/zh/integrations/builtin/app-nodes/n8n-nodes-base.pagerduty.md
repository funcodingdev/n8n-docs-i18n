# PagerDuty node <a href="#pagerduty-node" id="pagerduty-node"></a>

使用 PagerDuty node 自动化 PagerDuty 中的工作，并将 PagerDuty 与其他应用集成。n8n 内置支持大量 PagerDuty 功能，包括创建 incident note，以及更新和获取所有 log entry 与 user。

在本页中，你可以找到 PagerDuty node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [PagerDuty credentials](../credentials/pagerduty.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Incident
    * 创建 incident
    * 获取 incident
    * 获取所有 incident
    * 更新 incident
* Incident Note
    * 创建 incident note
    * 获取 incident 的所有 note
* Log Entry
    * 获取 log entry
    * 获取所有 log entry
* User
    * 获取 user
