# Drift node <a href="#drift-node" id="drift-node"></a>

使用 Drift node 自动化 Drift 中的工作，并将 Drift 与其他应用集成。n8n 内置支持大量 Drift 功能，包括创建、更新、删除和获取 contact。

在本页中，你可以找到 Drift node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Drift credentials](../credentials/drift.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Contact
    * 创建 contact
    * 获取 custom attribute
    * 删除 contact
    * 获取 contact
    * 更新 contact
