# Mailchimp node <a href="#mailchimp-node" id="mailchimp-node"></a>

使用 Mailchimp node 自动化 Mailchimp 中的工作，并将 Mailchimp 与其他应用集成。n8n 内置支持大量 Mailchimp 功能，包括创建、更新和删除 campaign，以及获取 list group。

在本页中，你可以找到 Mailchimp node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Mailchimp credentials](../credentials/mailchimp.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Campaign
    * 删除 campaign
    * 获取 campaign
    * 获取所有 campaign
    * 复制 campaign
    * 创建此 campaign 的 Resend to Non-Openers 版本
    * 发送 campaign
* List Group
    * 获取所有 group
* Member
    * 在 list 上创建新 member
    * 从 list 中删除 member
    * 获取 list 上的 member
    * 获取 list 上的所有 member
    * 更新 list 上的新 member
* Member Tag
    * 从 list member 添加 tag
    * 从 list member 移除 tag
