# PostHog node <a href="#posthog-node" id="posthog-node"></a>

使用 PostHog node 自动化 PostHog 中的工作，并将 PostHog 与其他应用集成。n8n 内置支持大量 PostHog 功能，包括创建 alias、event 和 identity，以及跟踪页面。

在本页中，你可以找到 PostHog node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [PostHog credentials](../credentials/posthog.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Alias
    * 创建 alias
* Event
    * 创建 event
* Identity
    * 创建
* Track
    * 跟踪 page
    * 跟踪 screen
