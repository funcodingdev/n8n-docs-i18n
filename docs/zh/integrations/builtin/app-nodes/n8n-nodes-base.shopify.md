# Shopify node <a href="#shopify-node" id="shopify-node"></a>

使用 Shopify node 自动化 Shopify 中的工作，并将 Shopify 与其他应用集成。n8n 内置支持大量 Shopify 功能，包括创建、更新、删除和获取 order 与 product。

在本页中，你可以找到 Shopify node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Shopify credentials](../credentials/shopify.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Order
    * 创建 order
    * 删除 order
    * 获取 order
    * 获取所有 order
    * 更新 order
* Product
    * 创建 product
    * 删除 product
    * 获取 product
    * 获取所有 product
    * 更新 product
