# Vero node <a href="#vero-node" id="vero-node"></a>

使用 Vero node 自动化 Vero 中的工作，并将 Vero 与其他应用集成。n8n 内置支持大量 Vero 功能，包括创建和删除用户。

在本页中，你可以找到 Vero node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Vero credentials](../credentials/vero.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* 用户
    * 创建或更新用户资料
    * 更改用户标识符
    * 取消订阅用户
    * 重新订阅用户
    * 删除用户
    * 向用户资料添加 tag
    * 从用户资料移除 tag
* 事件
    * 为特定客户跟踪事件
