# Bitwarden node <a href="#bitwarden-node" id="bitwarden-node"></a>

使用 Bitwarden node 自动化 Bitwarden 中的工作，并将 Bitwarden 与其他应用集成。n8n 内置支持大量 Bitwarden 功能，包括创建、获取、删除和更新 collection、event、group 与 member。

在本页中，你可以找到 Bitwarden node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [Bitwarden credentials](../credentials/bitwarden.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Collection
    * Delete
    * Get
    * Get All
    * Update
* Event
    * Get All
* Group
    * Create
    * Delete
    * Get
    * Get All
    * Get Members
    * Update
    * Update Members
* Member
    * Create
    * Delete
    * Get
    * Get All
    * Get Groups
    * Update
    * Update Groups
