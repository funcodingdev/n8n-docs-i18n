# AWS SES node <a href="#aws-ses-node" id="aws-ses-node"></a>

使用 AWS SES node 自动化 AWS SES 中的工作，并将 AWS SES 与其他应用集成。n8n 内置支持大量 AWS SES 功能，包括创建、获取、删除、发送、更新和添加 template 与 email。

在本页中，你可以找到 AWS SES node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

有关设置身份验证的指导，请参阅 [AWS SES credentials](../credentials/aws.md)。
{% endhint %}

## 操作 <a href="#operations" id="operations"></a>

* Custom Verification Email
    * 创建新的 custom verification email template
    * 删除现有 custom verification email template
    * 获取 custom email verification template
    * 获取 account 中所有现有 custom verification email template
    * 将 email address 添加到 identity 列表
    * 更新现有 custom verification email template
* Email
    * Send
    * Send Template
* Template
    * 创建 template
    * 删除 template
    * 获取 template
    * 获取所有 template
    * 更新 template
