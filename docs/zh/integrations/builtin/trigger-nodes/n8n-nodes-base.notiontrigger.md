# Notion Trigger

[Notion](https://notion.so) 是一个一体化工作区，可用于笔记、任务、wiki 和数据库。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/notion.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Notion Trigger integrations](https://n8n.io/integrations/notion-trigger-beta/) 页面。
{% endhint %}
