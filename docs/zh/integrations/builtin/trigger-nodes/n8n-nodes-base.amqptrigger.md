# AMQP Trigger node <a href="#amqp-trigger-node" id="amqp-trigger-node"></a>

[AMQP](https://www.amqp.org/) 是面向消息中间件的开放标准应用层协议。AMQP 的定义性特征包括面向消息、队列、路由、可靠性和安全性。此 node 支持兼容 AMQP 1.0 的消息 broker。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/amqp.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [AMQP integrations](https://n8n.io/integrations/amqp-trigger/) 页面。
{% endhint %}
