# Acuity Scheduling Trigger node <a href="#acuity-scheduling-trigger-node" id="acuity-scheduling-trigger-node"></a>

[Acuity Scheduling](https://acuityscheduling.com/) 是基于云的预约排程软件方案，让企业主能够在线管理预约。它可以根据用户时区自动同步日历，并就用户的预约安排发送定期 alert 和提醒。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/acuityscheduling.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Acuity Scheduling Trigger integrations](https://n8n.io/integrations/acuity-scheduling-trigger/) 页面。
{% endhint %}

## 事件 <a href="#events" id="events"></a>

* 预约已取消
* 预约已更改
* 预约已重新排程
* 预约已安排
* 订单已完成
