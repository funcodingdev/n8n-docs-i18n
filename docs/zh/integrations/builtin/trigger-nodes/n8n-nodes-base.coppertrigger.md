# Copper Trigger node <a href="#copper-trigger-node" id="copper-trigger-node"></a>

[Copper](https://www.copper.com/) 是一款注重与 Google Workspace 深度集成的 CRM。它主要面向中小型企业。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/copper.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Copper Trigger integrations](https://n8n.io/integrations/copper-trigger/) 页面。
{% endhint %}
