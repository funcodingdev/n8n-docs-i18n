# Jotform Trigger node <a href="#jotform-trigger-node" id="jotform-trigger-node"></a>

[Jotform](https://www.jotform.com/) 是在线表单构建服务。Jotform 的软件通过拖放式创建工具创建表单，并提供加密用户数据的选项。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/jotform.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Jotform Trigger integrations](https://n8n.io/integrations/jotform-trigger/) 页面。
{% endhint %}
