# GetResponse Trigger node <a href="#getresponse-trigger-node" id="getresponse-trigger-node"></a>

[GetResponse](https://www.getresponse.com/) 是一个在线平台，提供 email marketing 软件、landing page 创建器、webinar 托管等功能。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/getresponse.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [GetResponse Trigger integrations](https://n8n.io/integrations/getresponse-trigger/) 页面。
{% endhint %}

## 事件 <a href="#events" id="events"></a>

- 当客户订阅列表时接收通知
- 当客户从列表取消订阅时接收通知
- 当 email 被打开时接收通知
- 当 email 被点击时接收通知
- 当调查被提交时接收通知
