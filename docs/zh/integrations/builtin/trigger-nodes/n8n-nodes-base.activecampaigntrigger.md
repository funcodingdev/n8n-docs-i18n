# ActiveCampaign Trigger node <a href="#activecampaign-trigger-node" id="activecampaign-trigger-node"></a>

[ActiveCampaign](https://www.activecampaign.com/) 是面向中小型企业的云软件平台。该公司提供客户体验自动化软件，结合了 email marketing、marketing automation、sales automation 和 CRM 类别。
