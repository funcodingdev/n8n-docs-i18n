# Typeform Trigger node <a href="#typeform-trigger-node" id="typeform-trigger-node"></a>

[Typeform](https://www.typeform.com/) 是一家在线 SaaS 软件公司，专注于在线表单构建和在线调查。其主要软件会根据用户需求创建动态表单。
