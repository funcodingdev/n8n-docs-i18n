# Lemlist Trigger node <a href="#lemlist-trigger-node" id="lemlist-trigger-node"></a>

[Lemlist](https://Lemlist.com) 是一个 email outreach 平台，允许你自动生成个性化图片和视频，并发送个性化 cold email。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/lemlist.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Lemlist Trigger integrations](https://n8n.io/integrations/lemlist-trigger/) 页面。
{% endhint %}
