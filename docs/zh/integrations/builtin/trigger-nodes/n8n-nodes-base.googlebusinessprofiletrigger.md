# Google Business Profile Trigger node <a href="#google-business-profile-trigger-node" id="google-business-profile-trigger-node"></a>

使用 Google Business Profile Trigger node 响应 [Google Business Profile](https://www.google.com/business/) 中的事件，并将 Google Business Profile 与其他应用集成。n8n 内置支持响应新的 review。

在本页中，你可以找到 Google Business Profile Trigger node 可以响应的事件列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/google/README.md)找到此 node 的身份验证信息。
{% endhint %}
