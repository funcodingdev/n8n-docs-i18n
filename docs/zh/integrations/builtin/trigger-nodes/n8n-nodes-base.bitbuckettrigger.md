# Bitbucket Trigger node <a href="#bitbucket-trigger-node" id="bitbucket-trigger-node"></a>

[Bitbucket](https://bitbucket.org/) 是 Atlassian 旗下的基于 Web 的版本控制仓库托管服务，面向使用 Mercurial 或 Git 修订控制系统的源代码和开发项目。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/bitbucket.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Bitbucket Trigger integrations](https://n8n.io/integrations/bitbucket-trigger/) 页面。
{% endhint %}
