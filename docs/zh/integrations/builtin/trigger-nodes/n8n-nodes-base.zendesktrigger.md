# Zendesk Trigger node <a href="#zendesk-trigger-node" id="zendesk-trigger-node"></a>

[Zendesk](https://www.zendesk.com/) 是一个支持工单系统，旨在帮助跟踪、确定优先级并解决客户支持互动。Zendesk Support 不只是帮助台，还能通过任意渠道上的个性化、响应式支持来培育客户关系。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/zendesk.md)找到此 node 的身份验证信息。
{% endhint %}

{% endhint %}
