# Google Calendar Trigger

[Google Calendar](https://www.google.com/calendar/) 是 Google 开发的时间管理和日程安排日历服务。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/google/)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Google Calendar Trigger integrations](https://n8n.io/integrations/google-calendar-trigger/) 页面。
{% endhint %}
