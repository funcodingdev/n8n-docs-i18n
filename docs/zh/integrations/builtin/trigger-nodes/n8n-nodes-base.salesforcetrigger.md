# Salesforce Trigger node <a href="#salesforce-trigger-node" id="salesforce-trigger-node"></a>

使用 Salesforce Trigger node 响应 [Salesforce](https://login.salesforce.com/) 中的事件，并将 Salesforce 与其他应用程序集成。n8n 内置支持多种 Salesforce 事件。

在本页中，你可以找到 Salesforce Trigger node 可响应的事件列表，以及更多相关资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/salesforce.md)找到此 node 的身份验证信息。
{% endhint %}
{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Salesforce trigger integrations](https://n8n.io/integrations/salesforce-trigger/) 页面。
{% endhint %}

## 事件 <a href="#events" id="events"></a>

* Account 已创建
* Account 已更新
* Attachment 已创建
* Attachment 已更新
* Case 已创建
* Case 已更新
* Contact 已创建
* Contact 已更新
* Custom Object 已创建
* Custom Object 已更新
* Lead 已创建
* Lead 已更新
* Opportunity 已创建
* Opportunity 已更新
* Task 已创建
* Task 已更新
* User 已创建
* User 已更新
