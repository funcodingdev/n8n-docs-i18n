# Invoice Ninja Trigger node <a href="#invoice-ninja-trigger-node" id="invoice-ninja-trigger-node"></a>

[Invoice Ninja](https://www.invoiceninja.com/) 是面向自由职业者和企业的免费开源在线发票应用。它提供发票、付款、费用跟踪和时间任务功能。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/invoiceninja.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Invoice Ninja Trigger integrations](https://n8n.io/integrations/invoice-ninja-trigger/) 页面。
{% endhint %}
