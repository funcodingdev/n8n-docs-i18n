# MailerLite Trigger node <a href="#mailerlite-trigger-node" id="mailerlite-trigger-node"></a>

[MailerLite](https://www.mailerlite.com/) 是一个 email marketing 方案，为你提供易用的内容编辑器、简化的订阅者管理，以及包含关键统计信息的 campaign 报告。

在本页中，你可以找到 MailerLite Trigger node 可以响应的事件列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/mailerlite.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [MailerLite Trigger integrations](https://n8n.io/integrations/mailerlite-trigger/) 页面。
{% endhint %}

## 事件 <a href="#events" id="events"></a>

- Campaign 已发送
- 订阅者已添加到组
- 订阅者 automation 已完成
- 订阅者 automation 已触发
- 订阅者已退信
- 订阅者已创建
- 订阅者已投诉
- 订阅者已从组移除
- 订阅者取消订阅
- 订阅者已更新
