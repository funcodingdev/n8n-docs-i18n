# Postmark Trigger node <a href="#postmark-trigger-node" id="postmark-trigger-node"></a>

[Postmark](https://postmarkapp.com) 可帮助投递和跟踪应用邮件。你可以跟踪已发送或已处理的邮件数量、打开、退信和垃圾邮件投诉等统计信息。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/postmark.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Postmark Trigger integrations](https://n8n.io/integrations/postmark-trigger/) 页面。
{% endhint %}
