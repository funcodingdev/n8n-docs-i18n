# Help Scout Trigger node <a href="#help-scout-trigger-node" id="help-scout-trigger-node"></a>

[Help Scout](https://www.helpscout.com/) 是一款 help desk 软件，为客户服务专业人员提供基于 email 的客户支持平台、知识库工具，以及可嵌入的搜索/联系 widget。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/helpscout.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Help Scout Trigger integrations](https://n8n.io/integrations/helpscout-trigger/) 页面。
{% endhint %}
