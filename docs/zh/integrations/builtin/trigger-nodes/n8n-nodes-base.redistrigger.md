# Redis Trigger node <a href="#redis-trigger-node" id="redis-trigger-node"></a>

[Redis](https://redis.io/) 是一种开源内存数据结构存储，可用作数据库、缓存和消息代理。

使用 Redis Trigger node 订阅 Redis channel。当该 channel 收到新消息时，workflow 会启动。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/redis.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Redis Trigger integrations](https://n8n.io/integrations/redis-trigger/) 页面。
{% endhint %}
