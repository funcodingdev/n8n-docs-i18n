# MQTT Trigger node <a href="#mqtt-trigger-node" id="mqtt-trigger-node"></a>

[MQTT](https://mqtt.org) 是一种开放的 OASIS 和 ISO 标准轻量级发布-订阅网络协议，用于在设备之间传输消息。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/mqtt.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [MQTT Trigger integrations](https://n8n.io/integrations/mqtt-trigger/) 页面。
{% endhint %}
