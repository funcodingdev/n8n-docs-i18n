# Wufoo Trigger node <a href="#wufoo-trigger-node" id="wufoo-trigger-node"></a>

[Wufoo](https://wufoo.com) 是一个在线表单构建器，可帮助你无需编写代码即可创建自定义 HTML 表单。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/wufoo.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Wufoo Trigger integrations](https://n8n.io/integrations/wufoo-trigger/) 页面。
{% endhint %}
