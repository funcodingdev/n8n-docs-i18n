# Mailjet Trigger node <a href="#mailjet-trigger-node" id="mailjet-trigger-node"></a>

[Mailjet](https://www.mailjet.com/) 是基于云的 email 发送和跟踪系统。该平台允许专业人员发送 marketing email 和 transactional email。它包含用于设计 email、大规模发送和跟踪这些消息的工具。

**Credentials**

你可以在[这里](../credentials/mailjet.md)找到此 node 的身份验证信息。
{% endhint %}
