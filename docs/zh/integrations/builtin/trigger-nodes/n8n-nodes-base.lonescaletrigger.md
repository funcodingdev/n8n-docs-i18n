# LoneScale Trigger node <a href="#lonescale-trigger-node" id="lonescale-trigger-node"></a>

使用 LoneScale Trigger node 响应 [LoneScale](https://www.lonescale.com/) 中的 workflow 事件，并将 LoneScale 与其他应用集成。

在本页中，你可以找到 LoneScale node 支持的操作列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/lonescale.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [LoneScale Trigger integrations](https://n8n.io/integrations/lonescale-trigger/) 页面。
{% endhint %}
