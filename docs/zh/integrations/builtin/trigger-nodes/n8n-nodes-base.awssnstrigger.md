# AWS SNS Trigger node <a href="#aws-sns-trigger-node" id="aws-sns-trigger-node"></a>

[AWS SNS](https://aws.amazon.com/sns/) 是 Amazon Web Services 提供的通知服务。它为大规模消息投递提供低成本基础设施，主要面向移动用户。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/aws.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [AWS SNS Trigger integrations](https://n8n.io/integrations/aws-sns-trigger/) 页面。
{% endhint %}
