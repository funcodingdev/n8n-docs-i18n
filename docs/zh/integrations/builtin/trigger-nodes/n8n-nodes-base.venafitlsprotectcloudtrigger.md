# Venafi TLS Protect Cloud Trigger node <a href="#venafi-tls-protect-cloud-trigger-node" id="venafi-tls-protect-cloud-trigger-node"></a>

[Venafi](https://www.venafi.com/) 是一家网络安全公司，提供机器身份管理服务。他们提供的解决方案可管理和保护多种机器类型的身份，并提供全局可见性、生命周期自动化和可操作情报。

使用 n8n Venafi TLS Protect Cloud Trigger node，根据基于云的 [Venafi TLS Protect](https://vaas.venafi.com/) 服务中的事件启动 n8n workflow。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/venafitlsprotectcloud.md)找到此 node 的身份验证信息。
{% endhint %}

{% endhint %}
