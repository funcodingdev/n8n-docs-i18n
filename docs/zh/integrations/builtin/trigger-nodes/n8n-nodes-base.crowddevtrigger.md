# crowd.dev Trigger node <a href="#crowddev-trigger-node" id="crowddev-trigger-node"></a>

使用 crowd.dev Trigger node 响应 [crowd.dev](https://www.crowd.dev/) 中的事件，并将 crowd.dev 与其他应用集成。n8n 内置支持大量 crowd.dev 事件，包括新活动和新成员。

在本页中，你可以找到 crowd.dev Trigger node 可以响应的事件列表，以及更多资源链接。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/crowddev.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [crowd.dev Trigger integrations](https://n8n.io/integrations/crowddev-trigger/) 列表。
{% endhint %}

## 事件 <a href="#events" id="events"></a>

* 新活动
* 新成员

## 相关资源 <a href="#related-resources" id="related-resources"></a>

n8n 为 crowd.dev 提供了一个 app node。你可以在[这里](../app-nodes/n8n-nodes-base.crowddev.md)找到 node 文档。

在 n8n 网站上查看[示例 workflow 和相关内容](https://n8n.io/integrations/crowddev/)。

有关此服务的更多信息，请参阅 [crowd.dev 的文档](https://docs.crowd.dev/docs)。
