# Mautic Trigger node <a href="#mautic-trigger-node" id="mautic-trigger-node"></a>

[Mautic](https://www.mautic.org/) 是开源 marketing automation 软件，帮助在线业务自动化重复营销任务，例如 lead generation、contact scoring、contact segmentation 和 marketing campaign。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/mautic.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Mautic Trigger integrations](https://n8n.io/integrations/mautic-trigger/) 页面。
{% endhint %}
