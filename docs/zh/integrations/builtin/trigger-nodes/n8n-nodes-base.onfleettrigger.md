# Onfleet Trigger node <a href="#onfleet-trigger-node" id="onfleet-trigger-node"></a>

[Onfleet](https://onfleet.com/) 是一个物流平台，提供最后一公里配送解决方案。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/onfleet.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [Onfleet Trigger integrations](https://n8n.io/integrations/onfleet-trigger/) 页面。
{% endhint %}

## 事件 <a href="#events" id="events"></a>

在以下事件发生时触发 workflow：

* SMS 收件人选择退出
* SMS 收件人响应未收到
* 任务到达
* 任务已分配
* 任务已克隆
* 任务已完成
* 任务已创建
* 任务已延迟
* 任务 ETA
* 任务失败
* 任务已开始
* 任务已取消分配
* 任务已更新
* 工作人员已创建
* 工作人员已删除
* 工作人员值班
