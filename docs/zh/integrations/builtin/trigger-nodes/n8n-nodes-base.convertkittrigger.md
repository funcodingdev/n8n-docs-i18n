# ConvertKit Trigger node <a href="#convertkit-trigger-node" id="convertkit-trigger-node"></a>

[ConvertKit](https://www.convertkit.com/) 是功能完整的 email marketing 平台。使用 ConvertKit 构建 email 列表、发送 email broadcast、自动化序列、创建 segment，并构建 landing page。

{% hint style="info" %}
**Credentials**

你可以在[这里](../credentials/convertkit.md)找到此 node 的身份验证信息。
{% endhint %}

{% hint style="info" %}
**示例和模板**

有关帮助你入门的使用示例和模板，请参阅 n8n 的 [ConvertKit Trigger integrations](https://n8n.io/integrations/convertkit-trigger/) 页面。
{% endhint %}

## 事件 <a href="#events" id="events"></a>

* 表单订阅
* 链接点击
* 产品购买
* 购买已创建
* 购买已完成
* 序列已完成
* 序列订阅
* 订阅者已激活
* 订阅者取消订阅
* 添加 tag
* 移除 tag
