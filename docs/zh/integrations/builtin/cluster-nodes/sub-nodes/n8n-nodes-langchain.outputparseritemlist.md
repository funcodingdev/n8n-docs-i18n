# Item List Output Parser node <a href="#item-list-output-parser-node" id="item-list-output-parser-node"></a>

使用 Item List Output Parser node 返回具有特定长度和 separator 的 items 列表。

{% include "https://app.gitbook.com/s/GixZThfitWP21x2gQFpD/~/reusable/X6JM1Mgg5iwvZLDpGEB0/" %}

## Node 选项 <a href="#node-options" id="node-options"></a>

* **Number of Items**：输入要返回的最大 items 数量。设为 `-1` 表示不限制 items 数量。
* **Separator**：选择用于将结果拆分为单独 items 的 separator。默认为换行。
