# Embeddings Lemonade node <a href="#embeddings-lemonade-node" id="embeddings-lemonade-node"></a>

使用 Embeddings Lemonade node 通过 Lemonade server 托管和管理的 models 生成 vector embeddings。此 node 适用于执行 semantic search、clustering、similarity matching，或任何需要文本数值 vector 表示的任务的 workflows。

在本页中，你可以找到 Embeddings Lemonade node 支持的 operations 列表，以及更多相关资源链接。

{% endhint %}

## Node 参数 <a href="#node-parameters" id="node-parameters"></a>

使用以下参数配置此 node。

### Model <a href="#model" id="model"></a>

用于生成 embeddings 的 model。Models 通过为此 node 配置的 Lemonade server 加载和管理。请从你的 Lemonade instance 提供的可用 options 列表中选择所需 model。
