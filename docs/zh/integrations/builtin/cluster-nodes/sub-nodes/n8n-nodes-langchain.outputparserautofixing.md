# Auto-fixing Output Parser node <a href="#auto-fixing-output-parser-node" id="auto-fixing-output-parser-node"></a>

Auto-fixing Output Parser node 会包装另一个 output parser。如果第一个 parser 失败，它会调用另一个 LLM 来修复任何错误。
