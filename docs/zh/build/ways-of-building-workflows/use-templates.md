# 工作流模板 <a href="#workflow-templates" id="workflow-templates"></a>

创建新工作流时，你可以选择从空工作流开始，或使用现有模板[^1]。

模板提供：

* 入门帮助：n8n 可能已经有符合你需求的模板。
* 可构建内容的示例
* 创建你自己的工作流时的最佳实践

## 访问模板 <a href="#access-templates" id="access-templates"></a>

选择 <img src="../../../build/.gitbook/assets/templates.png" alt="View templates icon" data-size="line"> **Templates** 查看模板库。

如果你使用 n8n 的模板库，这会带你浏览 [n8n 网站上的 Workflows](https://n8n.io/workflows/)。如果你使用组织提供的自定义模板库，则可以在应用内搜索和浏览模板。
