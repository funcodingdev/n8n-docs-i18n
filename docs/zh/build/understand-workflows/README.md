# 工作流 <a href="#workflows" id="workflows"></a>

工作流[^1]是一组相互连接的 node，用于自动化一个流程。


* [创建](create-and-run-workflows.md)工作流。
* 使用[工作流模板](../ways-of-building-workflows/use-templates.md)帮助你快速开始。
* 了解 n8n 自动化中的关键[组成部分](workflow-components/README.md)。
* 使用 [Executions](understand-executions/README.md) 列表进行调试。
* 在用户之间[共享](../manage-workflows/share-with-others.md)工作流。

如果这是你第一次构建工作流，可以使用[快速入门指南](/try-it-out/index.md)快速试用 n8n 功能。

[^1]: n8n workflow 是一组用于自动化流程的 node。工作流会在触发条件发生时开始执行，并按顺序执行以完成复杂任务。
