# 让 AI 指定 tool 参数 <a href="#let-ai-specify-the-tool-parameters" id="let-ai-specify-the-tool-parameters"></a>

配置连接到 Tools Agent 的 tool[^1] 时，许多参数可以由 AI 模型自己填充。AI 模型会使用任务上下文以及其他已连接 tool 的信息，填入合适的详细内容。

有两种方式可以做到这一点，并且你可以在它们之间切换。

## 让模型填充参数 <a href="#let-the-model-fill-in-the-parameter" id="let-the-model-fill-in-the-parameter"></a>

tool 编辑对话框中每个适用的参数字段末尾都有一个额外按钮：

![图片显示参数字段右侧的星形图标](../../.gitbook/assets/ai-stars.png)

启用此按钮后，[AI Agent](#user-content-fn-2)[^2] 会为你填入 expression，无需进一步的用户输入。
该字段本身会填入一条消息，表示该参数已由模型自动定义。

如果想自己定义参数，请点击此框中的“X”以恢复为用户定义的值。请注意，`expression` 字段现在会包含此功能生成的 expression，不过你仍然可以继续编辑它，按下一节所述添加更多细节。

{% hint style="warning" %}
启用此功能会覆盖你可能已经添加的任何手动定义。
{% endhint %}

## 使用 `$fromAI()` 函数 <a href="#use-the-dollarfromai-function" id="use-the-dollarfromai-function"></a>

`$fromAI()` 函数使用 AI 为连接到 [Tools AI agent](https://app.gitbook.com/s/BKcbOzIWja8NfqKDcqHc/builtin/cluster-nodes/root-nodes/n8n-nodes-langchain.agent/tools-agent) 的 tool 动态填充参数。

{% hint style="info" %}
**仅适用于 tool**

`$fromAI()` 函数仅适用于连接到 AI Agent 节点的 tool。`$fromAI()` 函数不能用于 [Code](https://app.gitbook.com/s/BKcbOzIWja8NfqKDcqHc/builtin/cluster-nodes/sub-nodes/n8n-nodes-langchain.toolcode) tool，也不能用于[其他非 tool 的 cluster sub-node](https://app.gitbook.com/s/BKcbOzIWja8NfqKDcqHc/builtin/cluster-nodes/sub-nodes)。
{% endhint %}

要使用 `$fromAI()` 函数，请用必需的 `key` 参数调用它：

```javascript
{{ $fromAI('email') }}
```

`$fromAI()` 函数的 `key` 参数和其他参数不是对现有值的引用。你可以把这些参数理解为提示，AI 模型会用它们填充正确的数据。

例如，如果你选择名为 `email` 的 key，AI Model 会在其上下文、其他 tool 和输入数据中寻找电子邮件地址。在聊天工作流中，如果它无法在其他地方找到，可能会向用户询问电子邮件地址。你还可以选择传入 `description` 等其他参数，为 AI 模型提供额外上下文。

### 参数 <a href="#parameters" id="parameters"></a>

`$fromAI()` 函数接受以下参数：



| 参数 | 类型 | 是否必需？ | 说明 |
| --------- | ---- | --------- | ----------- |
| `key` | string | ✅ | 表示参数 key 或名称的字符串。长度必须在 1 到 64 个字符之间，并且只能包含小写字母、大写字母、数字、下划线和连字符。 |
| `description` | string | ❌ | 描述该参数的字符串。 |
| `type` | string | ❌ | 指定数据类型的字符串。可以是 string、number、boolean 或 json（默认为 string）。 |
| `defaultValue` | any | ❌ | 该参数使用的默认值。 |



### 示例 <a href="#examples" id="examples"></a>

例如，你可以使用以下 `$fromAI()` expression 动态用姓名填充字段：

```javascript
$fromAI("name", "The commenter's name", "string", "Jane Doe")
```

如果不需要可选参数，可以简化为：

```javascript
$fromAI("name")
```

要动态填充库存商品数量，可以使用如下 `$fromAI()` expression：

```javascript
$fromAI("numItemsInStock", "Number of items in stock", "number", 5)
```

如果你只想用模型提供的动态值填充字段的一部分，也可以在普通 expression 中使用它。例如，如果你希望模型为电子邮件填充 `subject` 参数，但始终在生成值前加上字符串 `Generated by AI:`，可以使用以下 expression：

```javascript
Generated by AI: {{ $fromAI("subject") }}
```

[^1]: 在 AI 语境中，tool 是一种附加资源，AI 在响应请求时可以引用它来获取特定信息或功能。AI 模型可以使用 tool 与外部系统交互，或完成特定且聚焦的任务。
[^2]: AI agent 是能够响应请求、做出决策，并为用户执行真实世界任务的人工智能系统。它们使用大型语言模型 (LLM) 解读用户输入，并根据可用的信息和资源决定如何最好地处理请求。
