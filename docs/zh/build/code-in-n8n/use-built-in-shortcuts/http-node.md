# HTTP node 变量 <a href="#http-node-variables" id="http-node-variables"></a>

使用 pagination 时，用于处理 HTTP node request 和 response 的变量。

关于如何使用 HTTP node（包括配置 pagination），请参阅 [HTTP Request](https://app.gitbook.com/s/BKcbOzIWja8NfqKDcqHc/builtin/core-nodes/n8n-nodes-base.httprequest)。

pagination 配置示例请参阅 [HTTP Request node cookbook | Pagination](../cookbook/http-request-node/pagination.md)。

{% hint style="info" %}
**仅 HTTP node**

这些变量用于 HTTP node 中的 expression。你不能在其他 node 中使用它们。
{% endhint %}
