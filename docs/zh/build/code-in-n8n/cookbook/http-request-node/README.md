# 使用 n8n HTTP Request node 的示例 <a href="#examples-using-n8ns-http-request-node" id="examples-using-n8ns-http-request-node"></a>

HTTP Request node 是 n8n 中用途最广的 node 之一。使用此 node 发起 HTTP request，以从任何具有 REST API 的 app 或 service 查询数据。

关于 node 设置的信息，请参阅 [HTTP Request](https://app.gitbook.com/s/BKcbOzIWja8NfqKDcqHc/builtin/core-nodes/n8n-nodes-base.httprequest)。
