# 管理 users <a href="#manage-users" id="manage-users"></a>

**Settings** > **Users** 页面会显示所有 users，包括 invitations pending 的 users。

## 删除 user <a href="#delete-a-user" id="delete-a-user"></a>

1. 打开要删除 user 的 **three-dot menu**，并选择 **Delete user**。
2. 确认你要删除该 user。
3. 如果他们是 active user，请选择是将其 workflow data 和 credentials 复制给 new user，还是永久删除其 workflows 和 credentials。

## 向 pending user 重新发送 invitation <a href="#resend-an-invitation-to-a-pending-user" id="resend-an-invitation-to-a-pending-user"></a>

点击 user 旁边的 menu icon，然后点击 **Resend invite**。
