# Security Assertion Markup Language (SAML) <a href="#security-assertion-markup-language-saml" id="security-assertion-markup-language-saml"></a>

{% include "https://app.gitbook.com/s/GixZThfitWP21x2gQFpD/~/reusable/9DXSTQsxYGisAV8xk6p4/" %}

本 section 说明如何在 n8n 中启用 SAML SSO (single sign-on)。它假设你熟悉 SAML。如果不熟悉，[SAML Explained in Plain English](https://www.onelogin.com/learn/saml) 可以帮助你理解 SAML 如何工作及其 benefits。

* [Set up SAML](set-up-saml.md)：在 n8n 中设置 SAML 的通用指南，并包含常见 IdPs 的 resources links。
* [Okta Workforce Identity SAML setup](set-up-okta-workforce-identity-saml.md)：配置 Okta 的 step-by-step guidance。
* [Azure AD SAML setup](set-up-azure-ad-saml.md)：使用 Azure AD 配置的 step-by-step guidance。
* [Troubleshooting](troubleshoot-saml.md)：遇到问题时可检查事项列表。
* [Managing users with SAML](manage-users-with-saml.md)：启用 SAML 后执行 user management tasks。
