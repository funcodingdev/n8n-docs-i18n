# User management <a href="#user-management" id="user-management"></a>

n8n 中的 user management 允许你邀请其他人到你的 n8n instance 中工作。它包括：

* Login 和 password management
* 添加和移除 users
* 三种 [account types](understand-account-types.md)：**Owner** 和 **Member**（Pro 与 Enterprise plans 还有 **Admin**）

{% hint style="info" %}
**Privacy**

User management feature 不会向 n8n 发送 personal information，例如 email 或 username。
{% endhint %}
