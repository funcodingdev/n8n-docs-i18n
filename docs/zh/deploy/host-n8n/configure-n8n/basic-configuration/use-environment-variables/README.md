# 环境变量概览 <a href="#environment-variables-overview" id="environment-variables-overview"></a>

本节列出 self-hosted n8n 时可用于更改 n8n 配置设置的环境变量。

如需完整且最新的环境变量列表，请查看 n8n docs 仓库中的完整参考：[Full environment variables index](https://github.com/n8n-io/n8n-docs/tree/main/docs/hosting/configuration/environment-variables)。

{% hint style="info" %}
**基于文件的配置**

你可以为 n8n 提供一个[配置文件](../../basic-configuration.md)。也可以在某些变量后追加 `_FILE`，通过单独的文件提供对应配置。
{% endhint %}
