# Deployment

{% include "https://app.gitbook.com/s/GixZThfitWP21x2gQFpD/~/reusable/ASsLuMLGKMy2O0q7awMF/" %}

本页列出 self-hosted n8n 实例的 deployment 配置选项，包括设置访问 URLs、启用 templates、自定义 encryption，以及配置 server details。

{% hint style="info" %}
**Proxy variable 优先级**

n8n 用来处理 proxy 环境变量（以 `_PROXY` 结尾的变量）的 [`proxy-from-env`](https://www.npmjs.com/package/proxy-from-env) package 会施加特定的变量优先级。尤其对 proxy 变量来说，当大小写版本同时存在时，小写版本（例如 `http_proxy`）优先于大写版本（例如 `HTTP_PROXY`）。

要了解更多 proxy 环境变量，请查看该 package details 中的 [environment variables section](https://www.npmjs.com/package/proxy-from-env#environment-variables)。
{% endhint %}

| Variable                               | Type                         | Default                                                          | Description                                                                                                                                                                                                                                                                                                                              |
| -------------------------------------- | ---------------------------- | ---------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `HTTP_PROXY`                           | String                       | -                                                                | 用于代理未加密 HTTP requests 的 URL。设置后，n8n 会通过该 proxy URL 代理 nodes 发出的所有未加密 HTTP traffic。                                                                                                                                                                                                   |
| `HTTPS_PROXY`                          | String                       | -                                                                | 用于代理 TLS/SSL 加密 HTTP requests 的 URL。设置后，n8n 会通过该 proxy URL 代理 nodes 发出的所有 TLS/SSL 加密 HTTP traffic。                                                                                                                                                                                       |
| `ALL_PROXY`                            | String                       | -                                                                | 用于代理未加密和加密 HTTP requests 的 URL。设置后，当更具体的变量（`HTTP_PROXY` 或 `HTTPS_PROXY`）不存在时，n8n 会使用此值。                                                                                                                                                          |
| `NO_PROXY`                             | String                       | -                                                                | 逗号分隔的 hostnames 或 URLs 列表，应绕过 proxy。使用 `HTTP_PROXY`、`HTTPS_PROXY` 或 `ALL_PROXY` 时，n8n 会直接连接这里定义的 URLs 或 hostnames，而不是使用 proxy。                                                                                                            |
| `N8N_ENFORCE_GLOBAL_USER_AGENT`        | Boolean                      | `false`                                                          | 设为 `true` 时，n8n 会在所有 outbound HTTP requests 中，将默认的裸 `n8n` User-Agent string 替换为符合 RFC 的值（`Mozilla/5.0 (compatible; n8n/<version>; +https://n8n.io/)`）。启用此项可防止 web application firewalls 阻止 n8n requests。                                                          |
| `N8N_GLOBAL_USER_AGENT_VALUE`          | String                       | -                                                                | 用于所有 outbound HTTP requests 的 custom User-Agent string。覆盖 `N8N_ENFORCE_GLOBAL_USER_AGENT` 设置的 RFC-compliant 默认值。当你不想向 upstream servers 暴露 n8n version 时很有用。                                                                                                                |
| `N8N_EDITOR_BASE_URL`                  | String                       | -                                                                | 用户可访问 editor 的 public URL。也用于 n8n 发送的 emails，以及 SAML based authentication 的 redirect URL。                                                                                                                                                                                                     |
| `N8N_DISABLE_UI`                       | Boolean                      | `false`                                                          | 设为 `true` 可禁用 UI。                                                                                                                                                                                                                                                                                                         |
| `N8N_PREVIEW_MODE`                     | Boolean                      | `false`                                                          | 设为 `true` 可在 preview mode 中运行。                                                                                                                                                                                                                                                                                                    |
| `N8N_TEMPLATES_ENABLED`                | Boolean                      | `true`                                                           | 启用 [workflow templates](#user-content-fn-1)[^1]（true）或禁用（false）。                                                                                                                                                                                                |
| `N8N_TEMPLATES_HOST`                   | String                       | `https://api.n8n.io`                                             | 如果创建自己的 workflow template library，请修改此项。请注意，要使用自己的 workflow templates library，你的 API 必须提供与 n8n 相同的 endpoints 和 response structure。更多信息请参考 [Workflow templates](https://app.gitbook.com/s/rPN1zU5jaYNvwH7RzxqA/ways-of-building-workflows/use-templates)。 |
| `N8N_ENCRYPTION_KEY`                   | String                       | Random key generated by n8n                                      | 提供用于加密 n8n database 中 credentials 的 custom key。默认情况下，n8n 会在首次启动时生成随机 key。                                                                                                                                                                                                             |
| `N8N_ENV_FEAT_ENCRYPTION_KEY_ROTATION` | Boolean                      | `false`                                                          | 在所有 instances（main 和 workers）上设为 `true`，以启用 [encryption key rotation](../../security/rotate-encryption-keys.md)。这是单向更改：请先完整备份 database。                                                                                                                                                      |
| `N8N_ENV_FEAT_OAUTH2_JWE`              | Boolean                      | `false`                                                          | 在所有 instances（main 和 workers）上设为 `true`，以启用 [JWE token decryption for OAuth 2.0 credentials](../../security/decrypt-oauth-20-tokens-with-jwe.md)。Preview feature。                                                                                                                                                       |
| `N8N_ENV_FEAT_TOKEN_EXCHANGE`          | Boolean                      | `false`                                                          | 设为 `true` 可启用 [token exchange for embedding partners](../../../../../../hosting/oem-deployment/token-exchange.md)。Preview feature。                                                                                                                                                                                            |
| `N8N_OAUTH_JWE_JWKS_PER_MINUTE`        | Number                       | `60`                                                             | public JWKS endpoint（`/rest/.well-known/jwks.json`）的 per-IP rate limit，该 endpoint 由 [JWE token decryption for OAuth 2.0 credentials](../../security/decrypt-oauth-20-tokens-with-jwe.md) 使用。                                                                                                                                              |
| `N8N_USER_FOLDER`                      | String                       | `user-folder`                                                    | 提供 n8n 创建 `.n8n` folder 的路径。此目录存储 user-specific data，例如 database file 和 encryption key。                                                                                                                                                                                            |
| `N8N_PATH`                             | String                       | `/`                                                              | n8n 部署到的 path。将 `N8N_PATH` 与 reverse proxies 组合使用可能导致 folder navigation issues。请使用 subdomain（例如 `n8n.example.com`），或在不使用 reverse proxy 时使用 `N8N_PATH`。                                                                                                                                         |
| `N8N_HOST`                             | String                       | `localhost`                                                      | n8n 运行所在的 host name。                                                                                                                                                                                                                                                                                                                   |
| `N8N_PORT`                             | Number                       | `5678`                                                           | n8n 运行使用的 HTTP port。                                                                                                                                                                                                                                                                                                               |
| `N8N_LISTEN_ADDRESS`                   | String                       | `::`                                                             | n8n 应监听的 IP address。                                                                                                                                                                                                                                                                                                     |
| `N8N_PROTOCOL`                         | Enum string: `http`, `https` | `http`                                                           | 用于访问 n8n 的 protocol。                                                                                                                                                                                                                                                                                                          |
| `N8N_SSL_KEY`                          | String                       | -                                                                | HTTPS protocol 的 SSL key。                                                                                                                                                                                                                                                                                                          |
| `N8N_SSL_CERT`                         | String                       | -                                                                | HTTPS protocol 的 SSL certificate。                                                                                                                                                                                                                                                                                                  |
| `N8N_PERSONALIZATION_ENABLED`          | Boolean                      | `true`                                                           | 是否询问用户 personalisation questions，然后据此自定义 n8n。                                                                                                                                                                                                                                                       |
| `N8N_VERSION_NOTIFICATIONS_ENABLED`    | Boolean                      | `true`                                                           | 启用后，n8n 会发送新版本和安全更新通知。                                                                                                                                                                                                                                                              |
| `N8N_VERSION_NOTIFICATIONS_ENDPOINT`   | String                       | `https://api.n8n.io/versions/`                                   | 用于获取 version information 的 endpoint。                                                                                                                                                                                                                                                                                      |
| `N8N_VERSION_NOTIFICATIONS_INFO_URL`   | String                       | `https://docs.n8n.io/getting-started/installation/updating.html` | New Versions panel 中显示的更多信息 URL。                                                                                                                                                                                                                                                                        |
| `N8N_DIAGNOSTICS_ENABLED`              | Boolean                      | `true`                                                           | 是否与 n8n 共享选定的匿名 [telemetry](https://app.gitbook.com/s/ukPPOMQ6NId4gpAIkPXa/)。请注意，如果你将其设为 `false`，将无法在 Code node 中启用 Ask AI。                                                                                                                                              |
| `N8N_DIAGNOSTICS_CONFIG_FRONTEND`      | String                       | `1zPn9bgWPzlQc0p8Gj1uiK6DOTn;https://telemetry.n8n.io`           | frontend 的 telemetry configuration。                                                                                                                                                                                                                                                                                                |
| `N8N_DIAGNOSTICS_CONFIG_BACKEND`       | String                       | `1zPn7YoGC3ZXE9zLeTKLuQCB4F6;https://telemetry.n8n.io/v1/batch`  | backend 的 telemetry configuration。                                                                                                                                                                                                                                                                                                 |
| `N8N_PUSH_BACKEND`                     | String                       | `websocket`                                                      | 选择 n8n backend 使用 server-sent events（`sse`）还是 WebSockets（`websocket`）向 UI 发送更改。                                                                                                                                                                                                                    |
| `VUE_APP_URL_BASE_API`                 | String                       | `http://localhost:5678/`                                         | 手动构建 `n8n-editor-ui` package 时使用，用于设置 frontend 如何访问 backend API。参考 [Configure the Base URL](../configuration-examples/configure-the-base-url.md)。                                                                                                                                       |
| `N8N_HIRING_BANNER_ENABLED`            | Boolean                      | `true`                                                           | 是否在 console 中显示 n8n hiring banner：`true` 表示显示，`false` 表示不显示。                                                                                                                                                                                                                                                              |
| `N8N_PUBLIC_API_SWAGGERUI_DISABLED`    | Boolean                      | `false`                                                          | 是否禁用 Swagger UI（API playground）：`true` 表示禁用，`false` 表示不禁用。                                                                                                                                                                                                                                                               |
| `N8N_PUBLIC_API_DISABLED`              | Boolean                      | `false`                                                          | 是否禁用 public API：`true` 表示禁用，`false` 表示不禁用。                                                                                                                                                                                                                                                                                 |
| `N8N_PUBLIC_API_ENDPOINT`              | String                       | `api`                                                            | public API endpoints 的 path。                                                                                                                                                                                                                                                                                                       |
| `N8N_GRACEFUL_SHUTDOWN_TIMEOUT`        | Number                       | `30`                                                             | n8n process 在退出前应等待 components 关闭的时长，单位为秒。                                                                                                                                                                                                                                |
| `N8N_DEV_RELOAD`                       | Boolean                      | `false`                                                          | 处理 n8n source code 时，设为 `true` 可在 source code files 发生更改时自动 reload 或 restart application。                                                                                                                                                                                  |
| `N8N_REINSTALL_MISSING_PACKAGES`       | Boolean                      | `false`                                                          | 如果设为 `true`，n8n 会自动尝试重新安装任何 missing packages。                                                                                                                                                                                                                                                      |
| `N8N_TUNNEL_SUBDOMAIN`                 | String                       | -                                                                | 指定 n8n tunnel 的 subdomain。如果未设置，n8n 会生成随机 subdomain。                                                                                                                                                                                                                                                |
| `N8N_PROXY_HOPS`                       | Number                       | 0                                                                | n8n 后面有多少个 reverse-proxies。                                                                                                                                                                                                                                                                                         |

[^1]: n8n templates 是由 n8n 和社区成员设计的预构建 workflows，你可以将其导入到 n8n 实例中。使用 templates 时，可能需要填写 credentials 并调整配置，以满足你的需求。
