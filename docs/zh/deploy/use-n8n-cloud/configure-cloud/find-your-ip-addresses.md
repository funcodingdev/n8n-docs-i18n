# Cloud IP 地址 <a href="#cloud-ip-addresses" id="cloud-ip-addresses"></a>

{% hint style="warning" %}
**Cloud IP 地址可能在无预警的情况下变化**

n8n 无法保证静态源 IP，因为 Cloud 运行在动态 cloud provider 环境中，并会按需求扩展基础设施。连接到 n8n 或从 n8n 连接出去时，你应该使用强身份验证和安全传输协议。
{% endhint %}
