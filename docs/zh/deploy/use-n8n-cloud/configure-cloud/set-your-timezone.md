# 设置 Cloud 实例时区 <a href="#set-the-cloud-instance-timezone" id="set-the-cloud-instance-timezone"></a>

你可以更改 n8n 实例的时区。这会影响 [Schedule Trigger](https://app.gitbook.com/s/BKcbOzIWja8NfqKDcqHc/builtin/core-nodes/n8n-nodes-base.scheduletrigger) 和 [Date & Time node](https://app.gitbook.com/s/BKcbOzIWja8NfqKDcqHc/builtin/core-nodes/n8n-nodes-base.datetime)。用户可以在 [Workflow settings](https://app.gitbook.com/s/rPN1zU5jaYNvwH7RzxqA/manage-workflows/configure-workflow-settings) 中为单个 workflows 配置时区。

1. 在 dashboard 上，选择 **Manage**。
1. 将 **Timezone** dropdown 更改为你想要的时区。
