# Cloud 管理后台 <a href="#cloud-admin-dashboard" id="cloud-admin-dashboard"></a>

实例所有者可以访问管理后台来管理自己的 Cloud 实例。你可以在这里升级 n8n 版本并设置时区。

## 从应用访问 dashboard <a href="#access-the-dashboard-from-the-app" id="access-the-dashboard-from-the-app"></a>

1. [登录 n8n](https://app.n8n.cloud/magic-link)
1. 选择 **Admin Dashboard**。n8n 会打开 dashboard。

## 应用离线时访问 dashboard <a href="#access-the-dashboard-if-the-app-is-offline" id="access-the-dashboard-if-the-app-is-offline"></a>

如果你的实例已宕机，仍然可以访问管理后台。当你登录应用时，n8n 会询问你是否需要用于访问 dashboard 的 magic link。选择 **Send magic link**，然后检查邮箱中的链接。
