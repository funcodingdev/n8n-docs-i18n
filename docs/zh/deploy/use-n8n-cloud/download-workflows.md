# 下载 workflows <a href="#download-workflows" id="download-workflows"></a>

n8n Cloud 实例所有者可以从最近一次 backup 下载 workflows。

你可以通过 [Cloud 管理后台](use-the-admin-dashboard.md)执行此操作。

## 如何下载 workflows <a href="#how-to-download-workflows" id="how-to-download-workflows"></a>

1. [登录 n8n](https://app.n8n.cloud/magic-link)。
1. 选择 **Admin Dashboard** 打开 dashboard。
1. 在 **Manage** 部分，选择 **Export** tab。
1. 选择 **Download Workflows**。

## 免费试用结束后访问 workflows <a href="#accessing-workflows-after-your-free-trial" id="accessing-workflows-after-your-free-trial"></a>

免费试用结束后，你有 **90 天** 可以下载 workflows。之后，所有 workflows 都会被 **永久删除**，且无法恢复。
