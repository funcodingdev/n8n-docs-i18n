# Cloud 免费试用 <a href="#cloud-free-trial" id="cloud-free-trial"></a>

创建新的 n8n cloud 试用后，你有 14 天可以试用 [Pro plan](https://n8n.io/pricing/) 的所有功能，包括：

- Global variables
- Insights dashboard
- Execution search
- 5 天 workflow history，可用于 rollback

试用提供 Pro plan 功能，但限制为 1000 次 executions，并使用与 [Starter plan](https://n8n.io/pricing/) 相同的计算能力。

## 升级到付费账号 <a href="#upgrade-to-a-paid-account" id="upgrade-to-a-paid-account"></a>

你可以随时升级到付费 n8n 账号。升级步骤：

1. 登录你的账号。
2. 点击右上角的 **Upgrade** 按钮。
3. 选择你的 plan，以及按年付费还是按月付费。
4. 选择 payment method。

## 试用到期 <a href="#trial-expiration" id="trial-expiration"></a>

如果你在试用结束前没有升级，试用会自动到期，你的 workspace 会被删除。

{% hint style="info" %}
**下载你的 workflows**

你可以[下载 workflows](download-workflows.md)，以后再复用。免费试用结束后，你有 90 天可以下载 workflows。
{% endhint %}

### 取消试用 <a href="#cancelling-your-trial" id="cancelling-your-trial"></a>

你不需要取消试用。试用会在试用期结束时自动到期，不会产生费用。你的所有数据会在不久后删除。

## Enterprise 试用 <a href="#enterprise-trial" id="enterprise-trial"></a>

如果你想测试 [Enterprise plan](https://n8n.io/pricing/)，可以联系销售团队。它包含如下功能：

- SSO SAML 和 LDAP
- 不同 environments
- External secret store integration
- Log streaming
- 使用 Git 进行 version control

点击 [n8n website](https://n8n.io/pricing/) 上的 **Contact** 按钮。
